# Chapter 01. 저장소 전체 지도와 학습 전략 / Repository Overview & Learning Strategy

---

## 🇰🇷 한국어

이 저장소는 **기초 ML → 시각화 → 앙상블 → 클러스터링 → SVM → 신경망 → NLP → OpenCV → 생성형 AI** 흐름으로 설계되어 있습니다.

### 핵심 목표

- 코드 한 파일 단위로 실행 가능한 실습 경험 확보
- 모델 결과를 "숫자"와 "시각"으로 동시에 이해
- 로컬 스크립트를 API 서비스 형태로 확장하는 사고 훈련

### 권장 학습 순서

1. `CrossValid.py` — 교차 검증 기초
2. `DecisionBoundary.py` — 경계 시각화
3. `RandomForest.py` — 앙상블
4. `KMeansClustering.py` — 비지도 클러스터링
5. `SVMClassifier.py` — 서포트 벡터 머신
6. `NeuralNetMLP.py` — 신경망
7. `LinearRegression.py` — 회귀
8. `SentimentAnalysis.py` — NLP
9. `OpenCVCPU.py` — 영상 처리
10. `HuggingFaceGPU.py` — 생성형 AI

## 🇺🇸 English

This repository is designed around the flow: **basic ML → visualization → ensembles → clustering → SVM → neural networks → NLP → OpenCV → generative AI**.

### Core Goals

- Hands-on experience with each concept in a single runnable file
- Understand model results both numerically and visually
- Practice extending local scripts to API-based services

### Recommended Learning Order

1. `CrossValid.py` — Cross-validation fundamentals
2. `DecisionBoundary.py` — Boundary visualization
3. `RandomForest.py` — Ensemble methods
4. `KMeansClustering.py` — Unsupervised clustering
5. `SVMClassifier.py` — Support Vector Machine
6. `NeuralNetMLP.py` — Neural networks
7. `LinearRegression.py` — Regression analysis
8. `SentimentAnalysis.py` — NLP text classification
9. `OpenCVCPU.py` — Video processing
10. `HuggingFaceGPU.py` — Generative AI
