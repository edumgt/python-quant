### 2. 트렌드 추종 전략 vs 평균 회귀 전략

**쉽게 이해하기**
- **트렌드 추종 전략 vs 평균 회귀 전략** 개념은 `입력(x)와 출력(y)의 관계 찾기`라고 생각하면 쉽습니다.
- 예측값만 보지 말고, 실제값과의 차이(오차)를 같이 확인하세요.

**미니 예시**
```python
# 트렌드 추종 전략 vs 평균 회귀 전략 미니 예시
x = [1, 2, 3, 4]
y = [2, 4, 6, 8]
slope = (y[-1] - y[0]) / (x[-1] - x[0])
next_y = y[-1] + slope
print('다음 값 예측:', next_y)
```

### 3. 시그널 생성 -> 주문 -> 리스크 관리 파이프라인

**쉽게 이해하기**
- **시그널 생성 -> 주문 -> 리스크 관리 파이프라인** 개념은 `자산 비중 정하기 → 성과 계산 → 비중 조정` 흐름으로 보면 쉽습니다.
- 한 번에 복잡하게 최적화하지 말고 2개 자산 예제로 먼저 감을 잡으세요.

**미니 예시**
```python
# 시그널 생성 -> 주문 -> 리스크 관리 파이프라인 미니 예시
weights = {'stock': 0.6, 'bond': 0.4}
asset_return = {'stock': 0.12, 'bond': 0.04}
portfolio_return = sum(weights[a] * asset_return[a] for a in weights)
print(f'기대수익률: {portfolio_return:.2%}')
```
