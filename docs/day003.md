## 천천히 배우기 (초등학생도 이해할 수 있어요)

### 1. 함수 정의와 호출 (def)

**쉽게 이해하기**
- **함수 정의와 호출 (def)** 개념은 작은 변수와 함수부터 직접 실행해보면 가장 빨리 익숙해집니다.
- 한 번에 길게 작성하기보다, 한 줄씩 출력 결과를 확인하면서 진행하세요.

**미니 예시**
```python
# 함수 정의와 호출 (def) 미니 예시
name = 'quant'
scores = [80, 90, 100]
avg = sum(scores) / len(scores)
print(f'{name} 평균 점수: {avg:.1f}')
```
