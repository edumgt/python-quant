### 5. 실습: 간단한 Transformer 시계열 예측 구현

**쉽게 이해하기**
- **실습: 간단한 Transformer 시계열 예측 구현** 개념은 `입력(x)와 출력(y)의 관계 찾기`라고 생각하면 쉽습니다.
- 예측값만 보지 말고, 실제값과의 차이(오차)를 같이 확인하세요.

**미니 예시**
```python
# 실습: 간단한 Transformer 시계열 예측 구현 미니 예시

y = [2, 4, 6, 8]
slope = (y[-1] - y[0]) / (x[-1] - x[0])
next_y = y[-1] + slope
print('다음 값 예측:', next_y)
```
