### 1. 객체지향 개념 이해 (클래스 vs 인스턴스)

**쉽게 이해하기**
- **객체지향 개념 이해 (클래스 vs 인스턴스)** 개념은 작은 변수와 함수부터 직접 실행해보면 가장 빨리 익숙해집니다.
- 한 번에 길게 작성하기보다, 한 줄씩 출력 결과를 확인하면서 진행하세요.

**미니 예시**
```python
# 객체지향 개념 이해 (클래스 vs 인스턴스) 미니 예시
name = 'quant'
scores = [80, 90, 100]
avg = sum(scores) / len(scores)
print(f'{name} 평균 점수: {avg:.1f}')
```

### 3. 속성(Attribute)과 메서드(Method)

**쉽게 이해하기**
- **속성(Attribute)과 메서드(Method)** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 속성(Attribute)과 메서드(Method) 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```
