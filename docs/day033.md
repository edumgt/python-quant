**미니 예시**
```python
# 실습: XGBoost로 금융 데이터 예측 및 튜닝 미니 예시

y = [2, 4, 6, 8]
slope = (y[-1] - y[0]) / (x[-1] - x[0])
next_y = y[-1] + slope
print('다음 값 예측:', next_y)
```
