## 천천히 배우기 (초등학생도 이해할 수 있어요)

### 1. 현대 포트폴리오 이론 (MPT) 개요

**쉽게 이해하기**
- **현대 포트폴리오 이론 (MPT) 개요** 개념은 `자산 비중 정하기 → 성과 계산 → 비중 조정` 흐름으로 보면 쉽습니다.
- 한 번에 복잡하게 최적화하지 말고 2개 자산 예제로 먼저 감을 잡으세요.

**미니 예시**
```python
# 현대 포트폴리오 이론 (MPT) 개요 미니 예시
weights = {'stock': 0.6, 'bond': 0.4}
asset_return = {'stock': 0.12, 'bond': 0.04}
portfolio_return = sum(weights[a] * asset_return[a] for a in weights)
print(f'기대수익률: {portfolio_return:.2%}')
```

### 2. 효율적 투자선(Efficient Frontier)

**쉽게 이해하기**
- **효율적 투자선(Efficient Frontier)** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 효율적 투자선(Efficient Frontier) 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 3. 성과 지표: CAGR, 변동성, MDD

**쉽게 이해하기**
- **성과 지표: CAGR, 변동성, MDD** 개념은 `일별 수익률 계산 → 누적 성과 확인 → 리스크 지표 비교`로 이해하면 쉽습니다.
- 먼저 작은 구간에서 계산이 맞는지 검증한 뒤, 전체 기간으로 확장하세요.

**미니 예시**
```python
# 성과 지표: CAGR, 변동성, MDD 미니 예시
returns = [0.01, -0.02, 0.015, 0.005]
cum = 1.0
for r in returns:
    cum *= (1 + r)
print(f'누적수익률: {(cum - 1):.2%}')
```

### 4. 위험 조정 지표: 샤프 비율, 소르티노 비율

**쉽게 이해하기**
- **위험 조정 지표: 샤프 비율, 소르티노 비율** 개념은 `일별 수익률 계산 → 누적 성과 확인 → 리스크 지표 비교`로 이해하면 쉽습니다.
- 먼저 작은 구간에서 계산이 맞는지 검증한 뒤, 전체 기간으로 확장하세요.

**미니 예시**
```python
# 위험 조정 지표: 샤프 비율, 소르티노 비율 미니 예시
returns = [0.01, -0.02, 0.015, 0.005]
cum = 1.0

    cum *= (1 + r)

```

### 5. 실습: 포트폴리오 성과 분석 대시보드
