## 천천히 배우기 (초등학생도 이해할 수 있어요)

### 1. ACF / PACF 시각화와 해석

**쉽게 이해하기**
- **ACF / PACF 시각화와 해석** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# ACF / PACF 시각화와 해석 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 2. AR, MA, ARMA 모델 개요

**쉽게 이해하기**
- **AR, MA, ARMA 모델 개요** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# AR, MA, ARMA 모델 개요 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 3. ARIMA(p, d, q) 파라미터 선택

**쉽게 이해하기**
- **ARIMA(p, d, q) 파라미터 선택** 개념은 `입력(x)와 출력(y)의 관계 찾기`라고 생각하면 쉽습니다.
- 예측값만 보지 말고, 실제값과의 차이(오차)를 같이 확인하세요.

**미니 예시**
```python
# ARIMA(p, d, q) 파라미터 선택 미니 예시
x = [1, 2, 3, 4]
y = [2, 4, 6, 8]
slope = (y[-1] - y[0]) / (x[-1] - x[0])
next_y = y[-1] + slope
print('다음 값 예측:', next_y)
```
