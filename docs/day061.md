## 학습 내용

### 1. PineScript로 이동평균 전략 구현

**쉽게 이해하기**
- **PineScript로 이동평균 전략 구현** 개념은 `지표 계산 → 조건 비교 → 신호 해석` 흐름으로 접근하면 쉽습니다.
- 지표 하나만 보지 말고 거래량/추세와 함께 확인하는 습관이 중요합니다.

**미니 예시**
```python
# PineScript로 이동평균 전략 구현 미니 예시
prices = [100, 102, 101, 105, 107]
ma3 = sum(prices[-3:]) / 3
signal = '매수 관찰' if prices[-1] > ma3 else '대기'
print('MA3:', round(ma3, 2), '| 신호:', signal)
```

### 2. PineScript 백테스트 결과 해석

**쉽게 이해하기**
- **PineScript 백테스트 결과 해석** 개념은 `일별 수익률 계산 → 누적 성과 확인 → 리스크 지표 비교`로 이해하면 쉽습니다.
- 먼저 작은 구간에서 계산이 맞는지 검증한 뒤, 전체 기간으로 확장하세요.

**미니 예시**
```python
# PineScript 백테스트 결과 해석 미니 예시
returns = [0.01, -0.02, 0.015, 0.005]
cum = 1.0
for r in returns:
    cum *= (1 + r)
print(f'누적수익률: {(cum - 1):.2%}')
```

### 3. Python 전략과 PineScript 결과 비교

**쉽게 이해하기**
- **Python 전략과 PineScript 결과 비교** 개념은 작은 변수와 함수부터 직접 실행해보면 가장 빨리 익숙해집니다.
- 한 번에 길게 작성하기보다, 한 줄씩 출력 결과를 확인하면서 진행하세요.

**미니 예시**
```python
# Python 전략과 PineScript 결과 비교 미니 예시
name = 'quant'
scores = [80, 90, 100]
avg = sum(scores) / len(scores)
print(f'{name} 평균 점수: {avg:.1f}')
```
