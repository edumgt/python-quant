### 1. 현대 포트폴리오 이론(MPT) 개요

> 📖 **Wikipedia**: [현대 포트폴리오 이론](https://ko.wikipedia.org/wiki/현대_포트폴리오_이론) · [분산 투자](https://ko.wikipedia.org/wiki/분산투자)

현대 포트폴리오 이론은 개별 자산을 따로 보는 것이 아니라, 자산을 함께 보유했을 때 전체 포트폴리오의 기대수익과 위험이 어떻게 달라지는지 분석합니다.

**핵심 아이디어**

- 자산의 수익률뿐 아니라 서로의 상관관계가 중요합니다.
- 상관관계가 낮은 자산을 섞으면 포트폴리오 변동성이 낮아질 수 있습니다.
- 좋은 포트폴리오는 같은 위험에서 더 높은 수익, 같은 수익에서 더 낮은 위험을 목표로 합니다.

> 📺 [🎬 현대 포트폴리오 이론 분산투자](https://www.youtube.com/results?search_query=현대+포트폴리오+이론+분산투자+한국어)

```python
weights = {"stock": 0.6, "bond": 0.4}
expected_returns = {"stock": 0.10, "bond": 0.04}

portfolio_return = sum(weights[a] * expected_returns[a] for a in weights)
print(f"포트폴리오 기대수익률: {portfolio_return:.2%}")
```

### 2. 효율적 투자선(Efficient Frontier)

> 📖 **Wikipedia**: [효율적 투자선](https://ko.wikipedia.org/wiki/효율적_프론티어)

효율적 투자선은 가능한 포트폴리오 중에서 위험 대비 수익이 가장 좋은 조합들의 집합입니다. 퀀트 실습에서는 여러 무작위 비중을 생성해 위험과 수익을 계산하고, 그중 우수한 조합을 찾는 방식으로 감을 잡습니다.

```python
import numpy as np

returns = np.array([0.10, 0.04])
cov = np.array([
    [0.20 ** 2, 0.20 * 0.08 * 0.2],
    [0.20 * 0.08 * 0.2, 0.08 ** 2],
])

for stock_weight in [0.0, 0.25, 0.5, 0.75, 1.0]:
    w = np.array([stock_weight, 1 - stock_weight])
    port_return = w @ returns
    port_vol = (w.T @ cov @ w) ** 0.5
    print(f"주식 {stock_weight:.0%}: 수익 {port_return:.2%}, 변동성 {port_vol:.2%}")
```

### 3. 성과 지표: CAGR, 변동성, MDD

> 📖 **Wikipedia**: [연평균 성장률](https://ko.wikipedia.org/wiki/연평균_성장률) · [변동성](https://ko.wikipedia.org/wiki/변동성_(금융))

| 지표 | 의미 | 해석 |
|------|------|------|
| CAGR | 연평균 복리 수익률 | 장기 성장 속도 |
| 변동성 | 수익률의 표준편차 | 흔들림의 크기 |
| MDD | 고점 대비 최대 하락률 | 투자자가 견뎌야 할 손실 폭 |
| 승률 | 수익이 난 기간의 비율 | 전략 안정성 보조 지표 |

```python
import pandas as pd

prices = pd.Series([100, 105, 98, 112, 120, 108, 130])
returns = prices.pct_change().dropna()
cumulative = prices / prices.iloc[0]

years = len(returns) / 252
cagr = cumulative.iloc[-1] ** (1 / years) - 1 if years > 0 else 0
vol = returns.std() * (252 ** 0.5)
mdd = (cumulative / cumulative.cummax() - 1).min()

print(f"CAGR: {cagr:.2%}")
print(f"변동성: {vol:.2%}")
print(f"MDD: {mdd:.2%}")
```

### 4. 위험 조정 지표: 샤프 비율, 소르티노 비율

> 📖 **Wikipedia**: [샤프 비율](https://ko.wikipedia.org/wiki/샤프_비율)

수익률만 높아도 위험이 지나치게 크면 좋은 전략이라고 보기 어렵습니다. 위험 조정 지표는 수익을 얻기 위해 감수한 변동성을 함께 평가합니다.

| 지표 | 계산 아이디어 | 특징 |
|------|---------------|------|
| 샤프 비율 | 초과수익률 ÷ 전체 변동성 | 가장 널리 쓰이는 위험 조정 지표 |
| 소르티노 비율 | 초과수익률 ÷ 하방 변동성 | 손실 방향 변동성만 반영 |
| 칼마 비율 | CAGR ÷ \|MDD\| | 최대 낙폭 대비 성과 |

```python
import numpy as np

daily_returns = np.array([0.01, -0.02, 0.015, 0.005, -0.004, 0.012])
risk_free_daily = 0.03 / 252
excess = daily_returns - risk_free_daily

sharpe = excess.mean() / daily_returns.std() * (252 ** 0.5)
downside = daily_returns[daily_returns < 0].std()
sortino = excess.mean() / downside * (252 ** 0.5)

print(f"Sharpe: {sharpe:.2f}")
print(f"Sortino: {sortino:.2f}")
```

### 5. 실습: 포트폴리오 성과 분석 대시보드

이번 실습의 목표는 여러 ETF의 가격 데이터를 가져와 포트폴리오 성과표를 만들고, 누적수익률·드로다운·월별 수익률을 시각화하는 것입니다.

```python
import yfinance as yf
import pandas as pd

tickers = ["SPY", "TLT", "GLD", "DBC"]
weights = pd.Series({"SPY": 0.5, "TLT": 0.3, "GLD": 0.15, "DBC": 0.05})

prices = yf.download(tickers, start="2018-01-01", auto_adjust=True)["Close"]
returns = prices.pct_change().dropna()
portfolio_returns = returns @ weights

cumulative = (1 + portfolio_returns).cumprod()
drawdown = cumulative / cumulative.cummax() - 1

summary = {
    "total_return": cumulative.iloc[-1] - 1,
    "annual_return": portfolio_returns.mean() * 252,
    "annual_vol": portfolio_returns.std() * (252 ** 0.5),
    "mdd": drawdown.min(),
}

print({k: f"{v:.2%}" for k, v in summary.items()})
```

# 주식투자자 유형별 추세 분석

한국 주식시장에서 투자자 유형은 주로 **개인(리테일)**, **기관(국내 기관)**, **외국인**으로 구분되며, 거래 행태, 선호 전략, 시장 영향력이 뚜렷합니다. 아래에서는 기존 분석에 **실적(수익률)**, **수급(순매수 동향)**, **차트 추세 영향**을 추가하여 종합 정리하였습니다. (2025~2026년 기준)

## 1. 개인 투자자 (Retail Investors)
- **거래 행태**: 전체 거래대금의 60~75% 차지. 단기 매매 성향 강하고 추세 반대(contrarian) 경향 두드러짐.
- **선호 전략**: 모멘텀·기술적 분석 중심, 성장주·테마주(반도체, 2차전지, 바이오 등) 선호. 단기 차익 실현 위주.
- **수급 동향**: 강세장에서 차익 실현 매도 우위. 최근(2026년) 외국인 매도 시 매수 주체로 유동성 공급 역할. 예: 코스피 급등 시 개인 순매수 확대.
- **실적(수익률)**: 외국인·기관 대비 상대적으로 낮음 (2025년 기준 개인 순매수 상위 종목 평균 수익률 약 88%, 외국인 201% 수준).
- **차트 추세 영향**: 단기 변동성 확대. 상승장에서 매도 압력, 하락장에서 저가 매수로 바닥 지지 역할. 고점 대비 차익 실현으로 조정 국면 촉발 가능.

## 2. 기관 투자자 (Institutional Investors: 연기금, 투신, 보험 등)
- **거래 행태**: 안정적·장기 투자 관점. 가치와 성장 혼합 전략.
- **선호 전략**: 기업 fundamentals(실적, 재무) 중시. 대형 우량주 중심 분산 투자.
- **수급 동향**: 중립적 또는 순매수 포지션 유지. 2026년 반도체·대형주 중심 순매수로 시장 안정화 기여. 최근 금융투자 중심으로 순매수 확대 사례 다수.
- **실적(수익률)**: 개인 대비 우수 (2025년 기관 순매수 상위 종목 평균 수익률 약 132%).
- **차트 추세 영향**: 지수 안정적 상승 추세 뒷받침. 대형주 중심 매수로 코스피 상승 모멘텀 강화. 변동성 완화 역할.

## 3. 외국인 투자자 (Foreign Investors)
- **거래 행태**: 모멘텀(추세 추종) 강함. 글로벌 관점 투자.
- **선호 전략**: 성장주·대형 우량주 중심. 한국 핵심 산업(반도체, 2차전지, 조선, 방산 등) 집중.
- **수급 동향**: 시장 방향성 주도. 2025~2026년 강세장에서 주요 매수 주체이나 차익 실현 시 대규모 순매도 발생. 보유 비중 약 29~30%. 최근 일부 기간 대규모 매도에도 지수 상승 지속 (개인·기관 매수로 상쇄).
- **실적(수익률)**: 가장 높음 (2025년 순매수 상위 종목 평균 수익률 약 201%). 삼성전자 등 대형주 집중 매수로 높은 초과 수익.
- **차트 추세 영향**: 상승 추세 주도. 반도체 사이클 호황 시 강한 매수로 신고가 랠리 견인. 매도 전환 시 단기 조정 유발하나, 전체적으로 가격 발견 기능 수행.

## 투자 스타일별 비교
- **가치 투자자**: 저평가 종목(PBR, PER) 장기 보유 — 기관·외국인 일부 강점.
- **성장 투자자**: 미래 성장 잠재력 중심 — 외국인과 일부 기관 우위.
- **모멘텀 투자자**: 단기 추세 추종 — 개인 투자자 중심, 상승장 강력하나 하락장 취약.

## 종합 시장 함의 및 2025~2026년 추세
- **수급 구조**: 개인이 유동성·변동성 공급, 외국인·기관이 방향성·가격 발견 주도. 2026년 들어 개인·ETF 중심 국내 유동성이 외국인 매도세를 상쇄하며 지수 상승 견인.
- **실적 환경**: 반도체 호황(AI 수요)으로 기업 영업이익 증가 전망. 외국인·기관이 실적 개선 종목 집중 매수로 우수 수익률 기록.
- **차트 추세**: 코스피 사상 최고치 경신(2026년 7,000~7,600대) 과정에서 외국인 매도에도 불구하고 개인·기관 매수로 상승 추세 지속. 반도체 중심 모멘텀 강세, 단기 조정 시 수급 공방전 심화 예상.

**2025~2026년 환경**에서는 글로벌 금리 사이클, 반도체·AI 모멘텀, 밸류업 정책이 투자자별 수익률 격차를 확대할 전망입니다. 개인 투자자는 해외 분산과 장기 투자 전환, 기관·외국인은 fundamentals 기반 접근 강화가 유효합니다.

# 초단타매매와 단타매매의 기준

초단타매매(스캘핑)와 단타매매(데이트레이딩)는 모두 **당일 매수 후 당일 매도**를 원칙으로 하는 단기 매매 기법입니다.  
보유 기간과 거래 빈도에서 큰 차이가 있습니다.

## 1. 초단타매매 (Scalping)

- **정의**: 매우 짧은 시간 안에 작은 가격 변동을 포착해 수익을 내는 매매 방식
- **보유 기간**: 수초 ~ 수분 (최대 5분 이내가 일반적)
- **거래 빈도**: 하루 수십 회 ~ 수백 회
- **주요 특징**:
  - 호가창의 미세한 움직임(스프레드) 이용
  - 1분 이하 또는 **틱 차트** 주로 사용
  - 목표 수익률: 거래당 0.3% ~ 1%
  - 고속 판단력과 집중력이 필수

## 2. 단타매매 (Day Trading)

- **정의**: 장중 가격 변동을 활용해 당일 또는 며칠 이내에 청산하는 매매
- **보유 기간**: 수분 ~ 수시간 (당일 청산이 원칙, 최대 1~3일 이내)
- **거래 빈도**: 하루 수회 ~ 십수 회
- **주요 특징**:
  - 1분, 3분, 5분 차트 활용
  - 뉴스, 거래량, 차트 패턴 종합 판단
  - 목표 수익률: 거래당 1% ~ 5%

## 비교 요약

| 항목            | 초단타매매 (Scalping)                | 단타매매 (Day Trading)               |
|-----------------|--------------------------------------|--------------------------------------|
| 보유 기간       | 수초 ~ 수분                          | 수분 ~ 수시간 (당일 청산)            |
| 거래 빈도       | 매우 높음 (수십~수백 회)             | 중간 (수회~십수 회)                  |
| 사용 차트       | 틱 차트, 1분 이하                    | 1~5분 차트                           |
| 수익 목표       | 작고 빈번 (0.3~1%)                   | 상대적으로 큼 (1~5%)                 |
| 집중도 / 스트레스 | 매우 높음                            | 높음                                 |
| 수수료 영향     | 매우 큼                              | 중간                                 |
| 적합한 투자자   | 전업, 빠른 판단력, 경험豊富          | 집중력 있음, 단기 트렌드 판단 가능   |

## 한국 주식시장 실전 기준

- **최적 시간대**: 오전 9:00 ~ 10:00 (장 개시 직후 변동성 최고)
- **종목 선정**: 거래대금 크고 변동성 높은 종목
- **위험 관리**: 손절 기준(-2~3%) 철저 준수, 레버리지 사용 주의
- **주의사항**: 높은 집중력 요구, 수수료와 슬리피지가 수익을 크게 잠식할 수 있음

**한 줄 요약**  
초단타는 **초고속 미세 수익**, 단타는 **단기 가격 변동 수익**을 노리는 전략으로,  
모두 높은 위험을 동반하므로 충분한 연습과 자본, 철저한 위험 관리가 필요합니다.
