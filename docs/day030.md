### 1. 로지스틱 회귀(Logistic Regression) 이론

**쉽게 이해하기**
- **로지스틱 회귀(Logistic Regression) 이론** 개념은 `입력(x)와 출력(y)의 관계 찾기`라고 생각하면 쉽습니다.
- 예측값만 보지 말고, 실제값과의 차이(오차)를 같이 확인하세요.

**미니 예시**
```python
# 로지스틱 회귀(Logistic Regression) 이론 미니 예시
x = [1, 2, 3, 4]
y = [2, 4, 6, 8]
slope = (y[-1] - y[0]) / (x[-1] - x[0])
next_y = y[-1] + slope
print('다음 값 예측:', next_y)
```

### 2. 분류 성능 지표 (Accuracy, Precision, Recall, F1, AUC)

**쉽게 이해하기**
- **분류 성능 지표 (Accuracy, Precision, Recall, F1, AUC)** 개념은 `특징 만들기 → 규칙 학습 → 라벨 예측` 3단계로 보면 쉽습니다.
- 모델 성능은 정확도 하나만 보지 말고 오분류 사례를 같이 점검하세요.

**미니 예시**
```python
# 분류 성능 지표 (Accuracy, Precision, Recall, F1, AUC) 미니 예시
scores = [0.2, 0.7, 0.9, 0.4]
labels = ['매도' if s < 0.5 else '매수' for s in scores]
print('분류 결과:', labels)
```

### 3. 교차 검증 (K-Fold Cross Validation)

**쉽게 이해하기**
- **교차 검증 (K-Fold Cross Validation)** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 교차 검증 (K-Fold Cross Validation) 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 4. 하이퍼파라미터 튜닝 (GridSearchCV)

**쉽게 이해하기**
- **하이퍼파라미터 튜닝 (GridSearchCV)** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 하이퍼파라미터 튜닝 (GridSearchCV) 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 5. 실습: 주가 상승/하락 방향 예측 분류 모델

**쉽게 이해하기**
- **실습: 주가 상승/하락 방향 예측 분류 모델** 개념은 `입력(x)와 출력(y)의 관계 찾기`라고 생각하면 쉽습니다.
- 예측값만 보지 말고, 실제값과의 차이(오차)를 같이 확인하세요.

**미니 예시**
```python
# 실습: 주가 상승/하락 방향 예측 분류 모델 미니 예시

y = [2, 4, 6, 8]
```

## 오늘 내용 한눈에 정리

- **로지스틱 회귀**는 어떤 일이 일어날 확률(예: 오를까/내릴까)을 예측하는 방법입니다.
- **정확도·정밀도·재현율·F1·AUC**는 모델이 얼마나 잘 맞추는지 보는 점수표입니다.
- **교차 검증**은 데이터를 여러 번 나눠서 공정하게 모델을 시험하는 방법입니다.
- **하이퍼파라미터 튜닝**은 모델 설정값을 바꿔 가장 좋은 조합을 찾는 과정입니다.

## 용어 사전 (초등학생도 이해하기 쉽게)

| 용어 | 한자 | 영어 | 쉬운 설명 |
|---|---|---|---|
| 회귀 | 回歸 | Regression | 숫자나 확률을 예측하는 계산 방법 |
| 분류 | 分類 | Classification | 정답을 여러 종류 중 하나로 고르는 것 |
| 정확도 | 正確度 | Accuracy | 전체 문제 중 맞춘 비율 |
| 정밀도 | 精密度 | Precision | 맞다고 한 것 중 진짜 정답 비율 |
| 재현율 | 再現率 | Recall | 진짜 정답을 얼마나 놓치지 않았는지 |
| 검증 | 檢證 | Validation | 모델이 진짜 잘 되는지 다시 확인하는 과정 |
