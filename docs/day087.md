## 천천히 배우기 (초등학생도 이해할 수 있어요)

### 1. 스토캐스틱(Stochastic Oscillator)

**쉽게 이해하기**
- **스토캐스틱(Stochastic Oscillator)** 개념은 `지표 계산 → 조건 비교 → 신호 해석` 흐름으로 접근하면 쉽습니다.
- 지표 하나만 보지 말고 거래량/추세와 함께 확인하는 습관이 중요합니다.

**미니 예시**
```python
# 스토캐스틱(Stochastic Oscillator) 미니 예시
prices = [100, 102, 101, 105, 107]
ma3 = sum(prices[-3:]) / 3
signal = '매수 관찰' if prices[-1] > ma3 else '대기'
print('MA3:', round(ma3, 2), '| 신호:', signal)
```

### 2. ATR(Average True Range) - 변동성 지표

**쉽게 이해하기**
- **ATR(Average True Range) - 변동성 지표** 개념은 `자산 비중 정하기 → 성과 계산 → 비중 조정` 흐름으로 보면 쉽습니다.
- 한 번에 복잡하게 최적화하지 말고 2개 자산 예제로 먼저 감을 잡으세요.

**미니 예시**
```python
# ATR(Average True Range) - 변동성 지표 미니 예시
weights = {'stock': 0.6, 'bond': 0.4}
asset_return = {'stock': 0.12, 'bond': 0.04}
portfolio_return = sum(weights[a] * asset_return[a] for a in weights)
print(f'기대수익률: {portfolio_return:.2%}')
```

### 3. OBV(On Balance Volume) - 거래량 지표

**쉽게 이해하기**
- **OBV(On Balance Volume) - 거래량 지표** 개념은 `지표 계산 → 조건 비교 → 신호 해석` 흐름으로 접근하면 쉽습니다.
- 지표 하나만 보지 말고 거래량/추세와 함께 확인하는 습관이 중요합니다.

**미니 예시**
```python
# OBV(On Balance Volume) - 거래량 지표 미니 예시
prices = [100, 102, 101, 105, 107]
ma3 = sum(prices[-3:]) / 3
signal = '매수 관찰' if prices[-1] > ma3 else '대기'
print('MA3:', round(ma3, 2), '| 신호:', signal)
```

### 4. CCI(Commodity Channel Index)

**쉽게 이해하기**
- **CCI(Commodity Channel Index)** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# CCI(Commodity Channel Index) 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 5. 실습: 다중 인디케이터 복합 전략 설계

**쉽게 이해하기**
- **실습: 다중 인디케이터 복합 전략 설계** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 실습: 다중 인디케이터 복합 전략 설계 미니 예시
```
