- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**

### 5. 실습: Python으로 DCF 모델 구현

**쉽게 이해하기**
- **실습: Python으로 DCF 모델 구현** 개념은 작은 변수와 함수부터 직접 실행해보면 가장 빨리 익숙해집니다.
- 한 번에 길게 작성하기보다, 한 줄씩 출력 결과를 확인하면서 진행하세요.

**미니 예시**
```python
# 실습: Python으로 DCF 모델 구현 미니 예시

scores = [80, 90, 100]
avg = sum(scores) / len(scores)
print(f'{name} 평균 점수: {avg:.1f}')
```
