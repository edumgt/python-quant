### 1. 경기 사이클(Expansion, Peak, Contraction, Trough) 이해

**쉽게 이해하기**
- **경기 사이클(Expansion, Peak, Contraction, Trough) 이해** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 경기 사이클(Expansion, Peak, Contraction, Trough) 이해 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 2. 경기 선행/동행/후행 지표

**쉽게 이해하기**
- **경기 선행/동행/후행 지표** 개념은 `지표 계산 → 조건 비교 → 신호 해석` 흐름으로 접근하면 쉽습니다.
- 지표 하나만 보지 말고 거래량/추세와 함께 확인하는 습관이 중요합니다.

**미니 예시**
```python
# 경기 선행/동행/후행 지표 미니 예시
prices = [100, 102, 101, 105, 107]
ma3 = sum(prices[-3:]) / 3
signal = '매수 관찰' if prices[-1] > ma3 else '대기'
print('MA3:', round(ma3, 2), '| 신호:', signal)
```
