### 5. 로그 수익률 변환

**쉽게 이해하기**
- **로그 수익률 변환** 개념은 `일별 수익률 계산 → 누적 성과 확인 → 리스크 지표 비교`로 이해하면 쉽습니다.
- 먼저 작은 구간에서 계산이 맞는지 검증한 뒤, 전체 기간으로 확장하세요.

**미니 예시**
```python
# 로그 수익률 변환 미니 예시
returns = [0.01, -0.02, 0.015, 0.005]
cum = 1.0
for r in returns:
    cum *= (1 + r)
print(f'누적수익률: {(cum - 1):.2%}')
```

### 2. AR, MA, ARMA 모델 개요

**쉽게 이해하기**
- **AR, MA, ARMA 모델 개요** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.


### 3. ARIMA(p, d, q) 파라미터 선택

**쉽게 이해하기**
- **ARIMA(p, d, q) 파라미터 선택** 개념은 `입력(x)와 출력(y)의 관계 찾기`라고 생각하면 쉽습니다.
- 예측값만 보지 말고, 실제값과의 차이(오차)를 같이 확인하세요.

**미니 예시**
```python
# ARIMA(p, d, q) 파라미터 선택 미니 예시
x = [1, 2, 3, 4]
y = [2, 4, 6, 8]
slope = (y[-1] - y[0]) / (x[-1] - x[0])
next_y = y[-1] + slope
print('다음 값 예측:', next_y)
```

### 4. SARIMA (계절성 ARIMA)

**쉽게 이해하기**
- **SARIMA (계절성 ARIMA)** 개념은 `입력(x)와 출력(y)의 관계 찾기`라고 생각하면 쉽습니다.
- 예측값만 보지 말고, 실제값과의 차이(오차)를 같이 확인하세요.

```python
# SARIMA (계절성 ARIMA) 미니 예시
```
