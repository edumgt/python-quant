### 1. 캔들 패턴 분석 (도지, 망치, 장악형 등)

**쉽게 이해하기**
- **캔들 패턴 분석 (도지, 망치, 장악형 등)** 개념은 복잡하게 외우기보다, `입력 데이터 → 규칙 적용 → 결과 확인` 흐름으로 보면 쉽습니다.
- 먼저 작은 샘플로 동작을 확인한 뒤, 실제 데이터로 확장하면 실수가 줄어듭니다.

**미니 예시**
```python
# 캔들 패턴 분석 (도지, 망치, 장악형 등) 미니 예시
sample = [100, 102, 101, 105]
change_rate = (sample[-1] - sample[0]) / sample[0]
print(f"변화율: {change_rate:.2%}")
```
