## 천천히 배우기 (초등학생도 이해할 수 있어요)

### 1. 캔들 패턴 분석 (도지, 망치, 장악형 등)

**쉽게 이해하기**
- **캔들 패턴 분석 (도지, 망치, 장악형 등)** 개념은 `지표 계산 → 조건 비교 → 신호 해석` 흐름으로 접근하면 쉽습니다.
- 지표 하나만 보지 말고 거래량/추세와 함께 확인하는 습관이 중요합니다.

**미니 예시**
```python
# 캔들 패턴 분석 (도지, 망치, 장악형 등) 미니 예시
prices = [100, 102, 101, 105, 107]
ma3 = sum(prices[-3:]) / 3
signal = '매수 관찰' if prices[-1] > ma3 else '대기'
print('MA3:', round(ma3, 2), '| 신호:', signal)
```

### 2. 차트 패턴 분석 (헤드앤숄더, 이중천장 등)

**쉽게 이해하기**
- **차트 패턴 분석 (헤드앤숄더, 이중천장 등)** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 차트 패턴 분석 (헤드앤숄더, 이중천장 등) 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 3. 엘리어트 파동 이론 개요
