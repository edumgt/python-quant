### 1. 소비자물가지수(CPI)와 생산자물가지수(PPI)

**쉽게 이해하기**
- **소비자물가지수(CPI)와 생산자물가지수(PPI)** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# CPI(소비자물가지수) 월별 변화 미니 예시
# 단위: 지수 (2020년 = 100 기준), 1월~4월
values = [104.5, 105.2, 104.8, 106.3]
delta = values[-1] - values[0]
print('시작 대비 변화:', round(delta, 2))
# → 1.8 상승: 4개월 만에 물가지수가 1.8포인트 올랐다는 뜻
```

### 2. 유가(WTI, 브렌트유) 분석

**쉽게 이해하기**
- **유가(WTI, 브렌트유) 분석** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# WTI 원유 주별 가격 변화 미니 예시
# 단위: 달러/배럴, 1주~4주
values = [75.2, 78.5, 76.8, 82.3]
delta = values[-1] - values[0]
print('시작 대비 변화:', round(delta, 2))
# → 7.1 상승: 한 달 만에 배럴당 7.1달러 올랐다는 뜻 (약 9.4% 상승)
```

### 3. 환율과 주가의 관계

**쉽게 이해하기**
- **환율과 주가의 관계** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 달러/원 환율 주별 변화 미니 예시
# 단위: 원(KRW), 1주~4주
values = [1320, 1338, 1325, 1356]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta, '원')
# → 36원 상승: 원화가 달러 대비 4주 만에 36원 약세가 됐다는 뜻
```

**delta(변화량)란?**

`delta = 끝값 - 시작값` 으로 구하는 가장 기본적인 변화 측정 방법이에요.

| delta 값 | 의미 |
|---|---|
| **양수(+)** | 환율 상승 → 원화 약세 (수입 물가 오름) |
| **음수(-)** | 환율 하락 → 원화 강세 (수입 물가 내림) |
| **0** | 변화 없음 |

> **왜 delta를 쓰나요?**
> 절대값(1356원)만 보면 "비싼지 싼지" 감이 없어요.
> 하지만 delta(+36원)를 보면 "한 달 만에 이만큼 변했다"는 **방향과 크기**를 한눈에 알 수 있어요.
> 경제지표 분석에서 delta는 추세 파악의 출발점이에요.

**왜 이름이 delta일까?**

그리스 문자 **Δ (Delta)** 는 수학에서 오래전부터 **"변화량"** 을 뜻하는 기호로 쓰여왔어요.

```
Δx = x₂ - x₁   (x의 변화량)
Δy = y₂ - y₁   (y의 변화량)
```

프로그래밍에서는 Δ를 변수명으로 직접 쓸 수 없어서 영어로 `delta` 라고 쓰게 됐어요.

실생활에서도 같은 뜻으로 쓰여요:
- **항공사 Delta** — 강 삼각주(삼각형 모양) 에서 유래, "큰 변화의 시작점"이라는 의미
- **코로나 델타 변이** — 기존 바이러스와 달라진(변화한) 변종이라서 델타
- **금융에서 "델타값"** — 옵션 가격이 기초자산 대비 얼마나 변하는지를 나타내는 지표

### 4. 실업률, GDP 성장률 지표 해석

**쉽게 이해하기**
- **실업률, GDP 성장률 지표 해석** 개념은 `지표 계산 → 조건 비교 → 신호 해석` 흐름으로 접근하면 쉽습니다.
- 지표 하나만 보지 말고 거래량/추세와 함께 확인하는 습관이 중요합니다.

**미니 예시**
```python
# 실업률, GDP 성장률 지표 해석 미니 예시
prices = [100, 102, 101, 105, 107]
ma3 = sum(prices[-3:]) / 3
signal = '매수 관찰' if prices[-1] > ma3 else '대기'
print('MA3:', round(ma3, 2), '| 신호:', signal)
```
