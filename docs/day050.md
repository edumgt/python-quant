## 천천히 배우기 (초등학생도 이해할 수 있어요)

### 1. 지지선(Support)과 저항선(Resistance)

**쉽게 이해하기**
- **지지선(Support)과 저항선(Resistance)** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 지지선(Support)과 저항선(Resistance) 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 2. 이동평균선(MA)과 골든크로스·데드크로스

**쉽게 이해하기**
- **이동평균선(MA)과 골든크로스·데드크로스** 개념은 `지표 계산 → 조건 비교 → 신호 해석` 흐름으로 접근하면 쉽습니다.
- 지표 하나만 보지 말고 거래량/추세와 함께 확인하는 습관이 중요합니다.

**미니 예시**
```python
# 이동평균선(MA)과 골든크로스·데드크로스 미니 예시
prices = [100, 102, 101, 105, 107]
ma3 = sum(prices[-3:]) / 3
signal = '매수 관찰' if prices[-1] > ma3 else '대기'
print('MA3:', round(ma3, 2), '| 신호:', signal)
```

### 6. 실습: Python으로 기술적 지표 계산 및 시각화
