## 천천히 배우기 (초등학생도 이해할 수 있어요)

### 1. 실시간 주가 조회 API 활용

**쉽게 이해하기**
- **실시간 주가 조회 API 활용** 개념은 `요청 정보 준비 → 호출 → 응답 확인` 순서로 보면 쉽습니다.
- 실제 호출 전에 endpoint, 헤더, 파라미터를 먼저 출력해 점검하면 오류를 줄일 수 있습니다.

**미니 예시**
```python
# 실시간 주가 조회 API 활용 미니 예시
endpoint = '/v1/orders'
token = 'Bearer <access_token>'  # 예시용: 실제 토큰은 환경변수/보안 저장소에서 읽으세요
headers = {'Authorization': token}
print('요청 준비:', endpoint, headers['Authorization'][:12] + '...')
```

### 2. 잔고 및 보유 종목 조회

**쉽게 이해하기**
- **잔고 및 보유 종목 조회** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 잔고 및 보유 종목 조회 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 4. 실습: 자동 주문 스크립트 기초 구현

**쉽게 이해하기**
- **실습: 자동 주문 스크립트 기초 구현**는 `작업 실행 → 로그 기록 → 상태 알림` 루프를 만들면 운영이 쉬워집니다.
- 먼저 로컬에서 한 번 자동 실행해보고, 실패 로그가 남는지 확인한 뒤 배포하세요.

**미니 예시**
```python
# 실습: 자동 주문 스크립트 기초 구현 미니 예시
job_name = 'rebalance-daily'
status = 'success'
log = f'[{job_name}] 상태: {status}'
print(log)
if status != 'success':
    print('알림 전송: 운영자 확인 필요')
```
