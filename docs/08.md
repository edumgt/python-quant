### 1. 시간에 따른 추이 시각화 (선 그래프)

**쉽게 이해하기**
- **시간에 따른 추이 시각화 (선 그래프)** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python

print('시작 대비 변화:', delta)
```

### 2. 캔들스틱 차트 구성 (mplfinance)

**쉽게 이해하기**
- **캔들스틱 차트 구성 (mplfinance)** 개념은 `지표 계산 → 조건 비교 → 신호 해석` 흐름으로 접근하면 쉽습니다.
- 지표 하나만 보지 말고 거래량/추세와 함께 확인하는 습관이 중요합니다.

**미니 예시**
```python
# 캔들스틱 차트 구성 (mplfinance) 미니 예시

ma3 = sum(prices[-3:]) / 3

print('MA3:', round(ma3, 2), '| 신호:', signal)
```
