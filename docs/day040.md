## 학습 내용

### 1. 회귀 지표: MAE, RMSE, MAPE

**쉽게 이해하기**
- **회귀 지표: MAE, RMSE, MAPE** 개념은 `입력(x)와 출력(y)의 관계 찾기`라고 생각하면 쉽습니다.
- 예측값만 보지 말고, 실제값과의 차이(오차)를 같이 확인하세요.

**미니 예시**
```python
# 회귀 지표: MAE, RMSE, MAPE 미니 예시
x = [1, 2, 3, 4]
y = [2, 4, 6, 8]
slope = (y[-1] - y[0]) / (x[-1] - x[0])
next_y = y[-1] + slope
print('다음 값 예측:', next_y)
```

### 2. 분류 지표: Confusion Matrix, ROC-AUC

**쉽게 이해하기**
- **분류 지표: Confusion Matrix, ROC-AUC** 개념은 `특징 만들기 → 규칙 학습 → 라벨 예측` 3단계로 보면 쉽습니다.
- 모델 성능은 정확도 하나만 보지 말고 오분류 사례를 같이 점검하세요.

**미니 예시**
```python
# 분류 지표: Confusion Matrix, ROC-AUC 미니 예시
scores = [0.2, 0.7, 0.9, 0.4]
labels = ['매도' if s < 0.5 else '매수' for s in scores]
print('분류 결과:', labels)
```

### 3. 과적합 진단 및 정규화 (L1, L2)

**쉽게 이해하기**
- **과적합 진단 및 정규화 (L1, L2)** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 과적합 진단 및 정규화 (L1, L2) 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 4. 학습률 스케줄러 및 조기 종료(Early Stopping)

**쉽게 이해하기**
- **학습률 스케줄러 및 조기 종료(Early Stopping)** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
