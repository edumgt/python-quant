### 1. 시계열(Time Series) 정의와 특징

**쉽게 이해하기**
- 시계열 데이터는 **시간 순서대로 기록된 값의 연속**입니다. 주가(종가), 코스피 지수, 원달러 환율, 국채 금리가 모두 시계열입니다.
- 일반 데이터와 달리 **인접 시점 간 상관관계**가 존재합니다 — 어제 주가는 오늘 주가 예측에 정보를 제공합니다.
- 금융 시계열의 기본 구조: **DatetimeIndex(날짜 인덱스)** + 가격·수익률·거래량 값

**미니 예시**
```python
import pandas as pd
import numpy as np

# 영업일(Business Day) 기준 날짜 인덱스 생성
dates = pd.date_range('2024-01-02', periods=5, freq='B')
prices = pd.Series([50000, 51000, 50500, 52000, 51500],
                   index=dates, name='Close')
print(prices)
print(f'\n인덱스 타입: {type(prices.index).__name__}')
print(f'기간 수익률: {(prices.iloc[-1] / prices.iloc[0] - 1):.2%}')
```

### 2. 추세(Trend), 계절성(Seasonality), 불규칙성(Noise)

**쉽게 이해하기**
- **추세(Trend)**: 장기적 방향성. 예) 코스피 10년 우상향, 장기 금리 하락
- **계절성(Seasonality)**: 주기적 반복 패턴. 예) 1월 효과(January Effect), 분기 실적 발표 패턴
- **불규칙성(Noise)**: 설명할 수 없는 무작위 요소 — 뉴스·이벤트에 의한 급등락
- **금융 활용**: 추세는 전략 방향, 계절성은 진입 타이밍, 노이즈는 리스크 관리에 각각 활용합니다

**미니 예시**
```python
import numpy as np
import pandas as pd

np.random.seed(42)
n = 120
t = np.arange(n)

# 합성 금융 시계열 = 추세 + 계절성 + 노이즈
trend       = 0.05 * t
seasonality = 3 * np.sin(2 * np.pi * t / 20)
noise       = np.random.randn(n) * 1.5
synthetic   = pd.Series(100 + trend + seasonality + noise, name='Price')

print(f'시작가: {synthetic.iloc[0]:.2f}')
print(f'종료가: {synthetic.iloc[-1]:.2f}')
print(f'추세 기여분: {trend[-1] - trend[0]:.2f}')
print(f'노이즈 표준편차: {noise.std():.2f}')
```

### 3. 가법 모형 vs 승법 모형

**쉽게 이해하기**
- **가법 모형** (Y = T + S + R): 변동 폭이 추세 수준과 무관. 금리·스프레드 등 절대값 중심 데이터에 적합
- **승법 모형** (Y = T × S × R): 추세가 높아질수록 변동 폭도 커짐. **주가·지수의 표준 모형**
- 로그 변환으로 승법 → 가법으로 변환 가능: `log(Y) = log(T) + log(S) + log(R)`
- 이것이 **퀀트 분석에서 로그 수익률을 사용하는 근본 이유**입니다

**미니 예시**
```python
import numpy as np

prices = [10000, 10300, 10100, 10600, 10800]

# 로그 수익률: 시간 가산성이 있어 합산으로 누적 수익률 계산 가능
log_returns = [np.log(prices[i] / prices[i-1]) for i in range(1, len(prices))]
print('로그 수익률:', [f'{r:.4f}' for r in log_returns])

total_log_r    = sum(log_returns)
total_simple_r = np.exp(total_log_r) - 1
print(f'로그 수익률 합산:  {total_log_r:.4f}')
print(f'누적 단순 수익률: {total_simple_r:.2%}')
```

### 4. 금융 시계열 데이터의 특수성 (비정상성, 팻테일, 변동성 군집)

**쉽게 이해하기**
- **비정상성(Non-stationarity)**: 주가는 시간에 따라 평균·분산이 변합니다. ARIMA 전에 차분(Differencing)으로 정상화가 필수입니다.
- **팻테일(Fat Tail)**: 주가 수익률의 극단적 등락은 정규분포 예상보다 훨씬 자주 발생합니다. 2008년 금융위기·2020년 코로나 급락이 사례입니다. 정규분포 가정 VaR는 리스크를 과소추정합니다.
- **변동성 군집(Volatility Clustering)**: 큰 변동 뒤에 큰 변동, 작은 변동 뒤에 작은 변동이 이어지는 패턴 — GARCH 모델이 이를 포착합니다.

**미니 예시**
```python
import numpy as np
from scipy import stats

np.random.seed(42)
n = 2000
normal_r   = np.random.normal(0, 0.01, n)
fat_tail_r = stats.t.rvs(df=4, scale=0.01, size=n)  # 자유도 4 → 강한 팻테일

print('=== 정규분포 수익률 ===')
print(f'초과첨도: {stats.kurtosis(normal_r):.2f}  (이론값 ≈ 0)')
print(f'1% VaR:  {np.percentile(normal_r, 1):.4f}')

print('\n=== 팻테일(t분포) 수익률 ===')
print(f'초과첨도: {stats.kurtosis(fat_tail_r):.2f}  (클수록 팻테일)')
print(f'1% VaR:  {np.percentile(fat_tail_r, 1):.4f}  ← 더 큰 손실 위험')
```

### 5. 실습: OHLCV 데이터 구조와 시계열 성분 파악

**쉽게 이해하기**
- **OHLCV**: Open(시가)·High(고가)·Low(저가)·Close(종가)·Volume(거래량) — 금융 분석의 기본 단위
- 퀀트 분석은 보통 **수정종가(Adj Close)** 기준 로그 수익률로 시작합니다 (배당·액면분할 반영)
- **연환산 공식**: 연간 수익률 = 일별 평균 × 252, 연간 변동성 = 일별 표준편차 × √252

**미니 예시**
```python
import pandas as pd
import numpy as np

np.random.seed(42)
n = 252
dates = pd.date_range('2024-01-02', periods=n, freq='B')
close = 50000 + np.cumsum(np.random.randn(n) * 400)

df = pd.DataFrame({
    'Open':   (close * np.random.uniform(0.998, 1.002, n)).round(0),
    'High':   (close * np.random.uniform(1.005, 1.015, n)).round(0),
    'Low':    (close * np.random.uniform(0.985, 0.995, n)).round(0),
    'Close':  close.round(0),
    'Volume': np.random.randint(1_000_000, 5_000_000, n),
}, index=dates)

df['LogReturn'] = np.log(df['Close']).diff()
ann_ret = df['LogReturn'].mean() * 252
ann_vol = df['LogReturn'].std()  * np.sqrt(252)
sharpe  = ann_ret / ann_vol

print(df[['Open', 'High', 'Low', 'Close', 'Volume']].tail(3))
print(f'\n연환산 수익률: {ann_ret:.2%}')
print(f'연환산 변동성: {ann_vol:.2%}')
print(f'샤프비율(rf=0): {sharpe:.2f}')
```

## 오늘 내용 한눈에 정리

- **시계열 데이터**는 시간 순서가 핵심 — DatetimeIndex를 반드시 설정해야 합니다.
- **금융 시계열**은 추세·계절성·노이즈가 혼재하며, 주가는 **승법 구조**가 일반적입니다.
- **로그 수익률**은 시간 가산성 덕분에 퀀트 분석의 표준 수익률 단위입니다.
- **비정상성·팻테일·변동성 군집**은 금융 시계열만의 특수한 통계적 성질입니다.
- OHLCV 수정종가 기반 로그 수익률 계산이 모든 분석의 출발점입니다.
