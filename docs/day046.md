## 학습 내용

### 1. 산업별 핵심 지표(KPI) 정리

**쉽게 이해하기**
- **산업별 핵심 지표(KPI) 정리** 개념은 `지표 계산 → 조건 비교 → 신호 해석` 흐름으로 접근하면 쉽습니다.
- 지표 하나만 보지 말고 거래량/추세와 함께 확인하는 습관이 중요합니다.

**미니 예시**
```python
# 산업별 핵심 지표(KPI) 정리 미니 예시
prices = [100, 102, 101, 105, 107]
ma3 = sum(prices[-3:]) / 3
signal = '매수 관찰' if prices[-1] > ma3 else '대기'
print('MA3:', round(ma3, 2), '| 신호:', signal)
```

### 2. 동종 업계 비교 분석 (Peer Comparison)

**쉽게 이해하기**
- **동종 업계 비교 분석 (Peer Comparison)** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 동종 업계 비교 분석 (Peer Comparison) 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

**쉽게 이해하기**

### 4. 실습: Python으로 산업별 주요 종목 재무 비교

**쉽게 이해하기**
- **실습: Python으로 산업별 주요 종목 재무 비교** 개념은 작은 변수와 함수부터 직접 실행해보면 가장 빨리 익숙해집니다.
- 한 번에 길게 작성하기보다, 한 줄씩 출력 결과를 확인하면서 진행하세요.

**미니 예시**
```python
# 실습: Python으로 산업별 주요 종목 재무 비교 미니 예시
name = 'quant'
scores = [80, 90, 100]
avg = sum(scores) / len(scores)
print(f'{name} 평균 점수: {avg:.1f}')
```
