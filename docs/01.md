퀀트·투자 분석을 하다 보면 기업의 **재무제표·세금·법적 구조**를 이해하는 것이 필수입니다. 기업이 왜 이런 회계처리를 하는지, 세금을 얼마나 내는지를 모르면 재무 분석이 왜곡됩니다.


## 들어가기 — 운용의 기본 개념

**운용(運用)**은 돈을 그냥 보관하는 것이 아니라, **투자를 통해 돈을 불리는 활동**입니다.

- 예금은 원금 보존 중심, 운용은 수익 창출 중심입니다.
- 운용의 핵심은 수익률을 높이면서도 손실 위험을 관리하는 균형입니다.
- 즉, 좋은 운용은 “많이 버는 것”과 “크게 잃지 않는 것”을 함께 추구합니다.

### Tenbagger(텐배거)란?

**Tenbagger**는 투자 원금이 **10배(10-bagger)**가 된 종목 또는 투자 성과를 뜻합니다.

- 100만 원이 1,000만 원이 되면 텐배거입니다.
- 장기 복리, 기업 성장, 분산 투자, 리스크 관리가 텐배거 가능성을 높입니다.
- 단기 급등만 노리기보다, 재무·산업·경쟁력을 함께 보는 분석이 중요합니다.

## A. 사람과 법인 — 법적 주체의 구분

> 📺 [🎬 사람과 법인 법적 주체의](https://www.youtube.com/results?search_query=사람과+법인+법적+주체의+구분+한국어)

### ① 자연인(個人) vs 법인(法人)

| 구분 | 자연인 (Natural Person) | 법인 (Legal Person / Corporation) |
|------|------------------------|-----------------------------------|
| 의미 | 태어나면서 권리·의무의 주체가 되는 사람 | 법이 사람처럼 권리·의무 능력을 부여한 **인위적 존재** |
| 사례 | 홍길동 씨 (개인) | 삼성전자 주식회사, 사단법인 XX협회 |
| 설립 방법 | 출생 | 설립 등기 (법원·등기소) |
| 사망/소멸 | 자연사 | 해산·청산 |
| 세금 | 소득세 (종합소득세·근로소득세) | 법인세 |
| 재산 | 본인 재산 = 개인 재산 | 법인 재산 ≠ 주주 재산 (분리) |
| 채무 책임 | 본인 전 재산으로 무한 책임 | 원칙적으로 법인 재산 한도 내 |

> 💡 법인이 **별도의 법적 인격**을 가진다는 것이 핵심입니다. 삼성전자가 빚을 져도, 이재용 회장이 개인 재산으로 갚을 의무는 없습니다(주주 유한 책임).

---

### ② 개인사업자 vs 법인사업자

사업을 할 때는 **개인사업자**와 **법인사업자(회사)** 중 하나를 선택합니다.

| 구분 | 개인사업자 | 법인사업자 |
|------|-----------|-----------|
| 법적 주체 | 자연인(사장 본인) | 별도 법인 |
| 설립 절차 | 사업자 등록 (세무서, 간단) | 법인 설립 등기 (법원) + 사업자 등록 |
| 최저 자본금 | 없음 | 없음 (단, 주식회사는 실무상 최소 100만원 이상) |
| 소득세 vs 법인세 | **종합소득세** (6~45% 누진) | **법인세** (9~24%, 낮은 세율) |
| 대표자 급여 | 비용 처리 **불가** (사장 소득 = 사업 소득) | 비용 처리 **가능** (급여로 처리) |
| 채무 책임 | **무한 책임** — 사업 빚 = 개인 빚 | **유한 책임** — 출자액 한도 |
| 대출·투자 유치 | 상대적으로 어려움 | 주식 발행·외부 투자 유치 용이 |
| 폐업 절차 | 간단 (폐업 신고) | 복잡 (청산·등기 말소) |
| 적합 상황 | 소규모 자영업·초기 1인 사업 | 성장을 목표로 한 사업, 투자 유치 필요 시 |

> 💡 **절세 전략**: 연 소득이 약 5,000만원을 초과하면 소득세 세율이 법인세보다 높아지기 시작합니다. 이 시점에 법인 전환을 검토하는 경우가 많습니다.

## B. 회사의 종류 — 책임 구조에 따른 분류

> 📺 [🎬 회사의 종류 책임 구조에](https://www.youtube.com/results?search_query=회사의+종류+책임+구조에+따른+분류+한국어)

상법은 회사를 **책임 방식**에 따라 5종으로 구분합니다.

| 회사 형태 | 사원의 책임 | 특징 | 실제 사용 |
|-----------|-----------|------|----------|
| **합명회사** | **전원 무한 책임** | 소수 신뢰 관계, 인적 회사 | 거의 없음 |
| **합자회사** | 무한책임사원 + 유한책임사원 혼합 | 투자자 보호 일부 가능 | 드묾 |
| **유한책임회사** | **전원 유한 책임** | 미국 LLC와 유사, 내부 자유도 높음 | 스타트업·VC |
| **유한회사** | **전원 유한 책임** | 사원 50인 이하, 비공개 | 중소기업 |
| **주식회사** | **전원 유한 책임** | 주식 발행, 공개·상장 가능 | **가장 보편적** |

### ① 무한책임 (Unlimited Liability)

사업체의 채무를 **개인 재산으로도 끝까지 갚아야** 하는 책임 구조입니다.

```
사업 빚 1억원 발생
  └── 무한책임 → 개인 통장·집·자동차까지 전부 상환 의무
```

- 해당: 개인사업자, 합명회사 사원, 합자회사 무한책임사원
- 위험: 사업 실패 시 개인 파산으로 이어질 수 있음

### ② 유한책임 (Limited Liability)

**출자한 금액 한도 내에서만** 책임지는 구조입니다.

```
주식 1,000만원어치 투자
  └── 유한책임 → 최대 손실 = 1,000만원 (개인 재산 추가 손실 없음)
```

- 해당: 주식회사 주주, 유한회사·유한책임회사 사원
- 의의: 위험 부담을 투자액으로 제한 → 대규모 자본 조달 가능

> 💡 유한책임 덕분에 수백만 명이 삼성전자 주식을 살 수 있습니다. 회사가 망해도 주가가 0원이 될 뿐, 개인이 회사 빚을 갚지 않아도 됩니다.

## C. 주식회사 (株式會社, Corporation)

> 📺 [🎬 주식회사](https://www.youtube.com/results?search_query=주식회사+한국어)

### ① 주식회사의 구조

```
주주 (출자자·소유자)
  │  출자 (주식 매입)
  ▼
주식회사 (법인)
  │  선임
  ▼
이사회 (Board of Directors)
  │  경영 위임
  ▼
대표이사 (CEO) + 경영진
```

| 기관 | 역할 |
|------|------|
| **주주총회** | 최고 의사결정 기관. 이사 선임, 합병, 정관 변경 등 |
| **이사회** | 경영 감독. 주요 업무 결의 |
| **대표이사** | 회사 대표, 일상 경영 집행 |
| **감사** | 이사 업무·회계 감시 |

### ② 주식 (株式, Share)

| 항목 | 내용 |
|------|------|
| 의미 | 주식회사 소유권(지분)을 균등하게 나눈 단위 |
| 종류 | 보통주 (의결권 O) / 우선주 (배당 우선, 의결권 X) |
| 액면가 | 정관에 기재된 주당 명목 금액 (보통 100원·500원·5,000원) |
| 시가 | 시장에서 거래되는 실제 가격 |
| 배당 | 이익 일부를 주주에게 분배 |
| 의결권 | 1주 = 1표 (보통주 기준) |

### ③ 상장 vs 비상장

| 구분 | 상장(公開)회사 | 비상장(私的)회사 |
|------|--------------|----------------|
| 주식 거래 | 거래소(KRX)에서 자유롭게 매매 | 장외·개인간 거래 |
| 공시 의무 | DART 공시 의무 (강함) | 거의 없음 |
| 자금 조달 | IPO·유상증자로 불특정 다수에게 | 제한적 |
| 대표 예시 | 삼성전자, 카카오 | 카카오 (상장 전), 비상장 중소기업 |

## D. 세금 — 개인과 법인의 차이

> 📺 [🎬 세금 개인과 법인의 차이](https://www.youtube.com/results?search_query=세금+개인과+법인의+차이+한국어)

### ① 소득세 (所得稅, Income Tax) — 개인

| 과세표준 | 세율 | 누진공제 |
|---------|------|---------|
| 1,400만원 이하 | **6%** | — |
| 1,400~5,000만원 | **15%** | 126만원 |
| 5,000~8,800만원 | **24%** | 576만원 |
| 8,800만원~1.5억원 | **35%** | 1,544만원 |
| 1.5억~3억원 | **38%** | 1,994만원 |
| 3억~5억원 | **40%** | 2,594만원 |
| 5억~10억원 | **42%** | 3,594만원 |
| 10억원 초과 | **45%** | 6,594만원 |

> 지방소득세 10% 추가 → 최고 세율 실질 **49.5%**

```python
def income_tax(taxable_income: int) -> int:
    """종합소득세 계산 (2024년 기준, 지방소득세 제외)"""
    brackets = [
        (14_000_000,   0.06,          0),
        (50_000_000,   0.15,  1_260_000),
        (88_000_000,   0.24,  5_760_000),
        (150_000_000,  0.35, 15_440_000),
        (300_000_000,  0.38, 19_940_000),
        (500_000_000,  0.40, 25_940_000),
        (1_000_000_000,0.42, 35_940_000),
        (float('inf'), 0.45, 65_940_000),
    ]
    for limit, rate, deduction in brackets:
        if taxable_income <= limit:
            return int(taxable_income * rate - deduction)
    return 0

for income in [30_000_000, 80_000_000, 200_000_000, 500_000_000]:
    tax = income_tax(income)
    print(f"소득 {income:>12,}원 → 세금 {tax:>11,}원 (실효세율 {tax/income*100:.1f}%)")
```

### ② 법인세 (法人稅, Corporate Tax) — 법인

| 과세표준 | 세율 |
|---------|------|
| 2억원 이하 | **9%** |
| 2억~200억원 | **19%** |
| 200억~3,000억원 | **21%** |
| 3,000억원 초과 | **24%** |

> 지방소득세 10% 추가 → 최고 실질 세율 **26.4%**

```python
def corporate_tax(taxable_income: int) -> int:
    """법인세 계산 (2024년 기준)"""
    brackets = [
        (200_000_000,       0.09,           0),
        (20_000_000_000,    0.19,  20_000_000),
        (300_000_000_000,   0.21,  60_000_000),
        (float('inf'),      0.24, 150_000_000),
    ]
    for limit, rate, deduction in brackets:
        if taxable_income <= limit:
            return int(taxable_income * rate - deduction)
    return 0
```

> 💡 **법인 전환 시점**: 과세표준이 약 5,000만원을 넘으면 소득세(24%)가 법인세(9~19%)보다 높아지기 시작합니다.

## E. 세무 기초

> 📺 [🎬 세무 기초](https://www.youtube.com/results?search_query=세무+기초+한국어)

### ① 세금 분류 체계

```
세금
├── 국세 (국가)
│   ├── 직접세: 소득세, 법인세, 상속세, 증여세
│   └── 간접세: 부가가치세(VAT), 개별소비세
└── 지방세 (지자체)
    ├── 취득세, 재산세
    └── 지방소득세 (소득세·법인세의 10%)
```

### ② 부가가치세 (VAT)

| 항목 | 내용 |
|------|------|
| 세율 | **10%** |
| 납부액 | 매출세액 − 매입세액 |
| 신고 주기 | 연 2회 (1월, 7월) |

```python
sales, purchase = 10_000_000, 6_000_000
vat = sales * 0.1 - purchase * 0.1
print(f"납부 부가세: {vat:,.0f}원")   # 400,000원
```

### ③ 원천징수 세율

| 소득 종류 | 세율 |
|-----------|------|
| 이자·배당소득 | **15.4%** |
| 프리랜서(사업소득) | **3.3%** |
| 근로소득 | 간이세액표 |

## E-A. 조세의 분류

> 📺 [🎬 조세의 분류](https://www.youtube.com/results?search_query=조세의+분류+한국어)

조세(租稅)는 **국가 또는 지방자치단체가 재정 수입을 위해 법률에 따라 강제적으로 징수하는 금전급부**입니다. 분류 기준에 따라 다양하게 나뉩니다.

---

### ① 과세 주체(부과 주체)에 따른 분류

세금을 **누가 부과·징수하느냐**에 따라 나눕니다.

| 구분 | 정의 | 주요 세목 |
|------|------|-----------|
| **국세 (國稅)** | 중앙정부(국가)가 부과·징수 | 소득세, 법인세, 부가가치세, 상속세, 증여세, 개별소비세, 종합부동산세 |
| **지방세 (地方稅)** | 지방자치단체(시·도, 시·군·구)가 부과·징수 | 취득세, 재산세, 지방소득세, 지방교육세, 주민세 |

```
세금
├── 국세 (국가 → 세무서·국세청)
│   ├── 내국세: 소득세, 법인세, 부가가치세, 상속·증여세 등
│   └── 관세: 수입품에 부과
└── 지방세 (지자체 → 시청·구청)
    ├── 도세: 취득세, 지방소비세
    └── 시·군세: 재산세, 주민세, 지방소득세
```

> 💡 지방소득세는 소득세·법인세 산출세액의 **10%**를 별도로 납부합니다. 예: 소득세 1,000만 원 → 지방소득세 100만 원 추가.

---

### ② 전가(轉嫁) 여부에 따른 분류 — 직접세 vs 간접세

세금 부담이 **납세자 본인에게 귀착되느냐, 타인에게 전가되느냐**에 따라 나눕니다.

| 구분 | 정의 | 특징 | 주요 세목 |
|------|------|------|-----------|
| **직접세 (Direct Tax)** | 납세자(세금 내는 사람) = 담세자(실제 부담자) | 소득·재산에 비례, 누진 적용 가능 | 소득세, 법인세, 상속세, 증여세, 종합부동산세 |
| **간접세 (Indirect Tax)** | 납세자 ≠ 담세자 (판매자가 내지만 소비자가 부담) | 물가에 포함되어 최종 소비자가 실질 부담 | 부가가치세(VAT), 개별소비세, 주세, 담배소비세 |

```
[직접세 예시] 소득세
  월급쟁이 → 소득세 원천징수 → 국가
  납세자 = 담세자 (본인 부담)

[간접세 예시] 부가가치세(VAT)
  소비자 → 상품 구매 시 VAT 포함 가격 지불
    → 판매자(납세자)가 국가에 납부
  납세자(판매자) ≠ 담세자(소비자)
```

> 💡 간접세는 소득 수준에 관계없이 동일한 금액을 부담하므로 **역진성**(저소득자에게 상대적으로 부담이 큰 현상)이 나타납니다.

> ✅ **금융소득(이자·배당)은 직접세**로 분류되며, 기본적으로 **이자소득세/배당소득세 14% + 지방소득세 1.4%(총 15.4%)**가 원천징수됩니다.  
> 연간 금융소득(이자+배당) 합계가 **2,000만 원 초과**하면 종합소득에 합산해 종합과세합니다.

### ③ 과세 대상(과세 물건)에 따른 분류

**무엇에 세금을 매기느냐**에 따라 나눕니다.

| 과세 유형 | 정의 | 세목 예시 |
|-----------|------|-----------|
| **소득세 계열** | 소득(벌어들인 돈)에 부과 | 소득세, 법인세, 지방소득세 |
| **재산세 계열** | 보유 재산(자산)에 부과 | 재산세, 종합부동산세, 상속세, 증여세 |
| **소비세 계열** | 소비(지출·구매)에 부과 | 부가가치세, 개별소비세, 주세, 담배소비세 |
| **유통세 계열** | 재산·권리의 이전(거래)에 부과 | 취득세, 인지세, 증권거래세 |

```
투자자 관련 세금 흐름 예시
  주식 매입 → 증권거래세 (유통세)
  주식 보유 → 배당소득세 (소득세)
  주식 매도 차익 → 금융투자소득세/양도소득세 (소득세)
  부동산 취득 → 취득세 (유통세)
  부동산 보유 → 재산세·종부세 (재산세)
  부동산 양도 → 양도소득세 (소득세)
```

### ④ 세율 구조에 따른 분류

**세율이 어떻게 적용되느냐**에 따라 나눕니다.

| 구분 | 정의 | 특징 | 예시 |
|------|------|------|------|
| **누진세 (Progressive Tax)** | 과세표준이 클수록 높은 세율 적용 | 소득 재분배 효과 | 소득세 (6~45%), 법인세 (9~24%) |
| **비례세 (Proportional Tax)** | 과세표준에 관계없이 동일 세율 | 단순·명확 | 부가가치세 (10%), 취득세 (4%) |
| **역진세 (Regressive Tax)** | 과세표준이 커질수록 실효세율 하락 | 저소득층 상대적 부담 증가 | 일부 간접세 (소득 대비) |

```python
# 누진세 계산 예시 — 2024년 기준 소득세
def income_tax(income):
    brackets = [
        (14_000_000, 0.06),
        (50_000_000, 0.15),
        (88_000_000, 0.24),
        (150_000_000, 0.35),
        (300_000_000, 0.38),
        (500_000_000, 0.40),
        (1_000_000_000, 0.42),
        (float('inf'), 0.45),
    ]
    tax, prev = 0, 0
    for limit, rate in brackets:
        if income <= prev:
            break
        taxable = min(income, limit) - prev
        tax += taxable * rate
        prev = limit
    return tax

print(f"5천만원 소득세: {income_tax(50_000_000):,.0f}원")   # 약 624만원
print(f"1억원 소득세:   {income_tax(100_000_000):,.0f}원")  # 약 2,010만원
```

```
    → 세무서에 납부 (다음달 10일까지)
    → 직원은 연말정산으로 정산
```

### ① 월급쟁이의 세금 — 근로소득세와 연말정산

직장인의 세금은 **회사가 매월 급여에서 미리 원천징수**하고, 연말에 실제 세액과 비교해 **환급 or 추가납부**합니다.

```
[월급 흐름]
  총급여
  − 비과세(식대 월 20만원, 차량유지비 월 20만원 등)
  = 과세 급여
  − 각종 공제 (인적공제, 신용카드, 의료비, 교육비, 기부금 등)
  = 과세표준
  × 세율(6~45% 누진)
  = 산출세액
  − 세액공제
  = 결정세액
  ↔ 기납부세액(원천징수 합계) 비교
  → 환급 or 추가납부
```

**핵심 공제 항목 요약**

| 공제 항목 | 한도·조건 | 절세 팁 |
|-----------|-----------|---------|
| **신용카드·체크카드** | 총급여 25% 초과분의 15~40% 공제 | 체크카드·현금영수증이 공제율 높음 |
| **의료비** | 총급여 3% 초과분의 15% (본인·장애인은 한도 없음) | 본인 의료비는 한도 없이 공제 |
| **교육비** | 본인 전액, 자녀 1인당 300만원 | 학원비는 제외(학교·교육기관만) |
| **보험료** | 보장성보험 연 100만원 한도 15% 공제 | 자동차보험, 실손보험 포함 |
| **연금저축·IRP** | 연금저축 600만원 + IRP 합산 900만원 한도, 세액공제 13~16.5% | 소득이 낮을수록 공제율 높음 |
| **기부금** | 법정·지정 기부금 15~30% 공제 | 고향사랑기부제: 10만원 이하 전액 세액공제 |

> 💡 **13월의 월급**: 공제 항목을 잘 챙기면 수십만~수백만원 환급 가능. 반대로 소득공제 누락 시 추가납부가 발생합니다.

### ② 프리랜서·부업 소득의 세금

| 구분 | 원천징수 | 신고 의무 | 비고 |
|------|----------|-----------|------|
| **프리랜서 소득** | 3.3% 원천징수 (소득세 3% + 지방소득세 0.3%) | 5월 종합소득세 신고 | 필요경비 인정 가능 |
| **유튜브·블로그 수익** | 별도 원천징수 없음 | 5월 종합소득세 신고 | 연 100만원 초과 시 신고 의무 |
| **직장인 부업 소득** | 프리랜서와 동일 | 근로소득 + 사업소득 합산 신고 | 연 300만원 초과 시 합산 |
| **중고 거래** | 없음 | 반복·영리 목적 시 신고 대상 | 생활용품 단순 처분은 비과세 |

```
[프리랜서 절세 포인트]
  수입 100만원 × 3.3% = 3.3만원 원천징수
  → 5월 신고 시 필요경비(장비, 교통비, 통신비 등) 공제
  → 실제 납부 세금 줄어들 수 있음 (환급 가능)
```

### ③ 소비할 때 내는 세금 — 부가가치세(VAT)

우리가 물건을 살 때 이미 가격에 포함된 세금입니다.

```
커피 한 잔 5,000원
  → 실제 가격 4,545원 + VAT 455원(10%)
  → 가게 사장이 매 분기 국세청에 455원 납부
  → 소비자는 자신도 모르게 세금을 낸 것
```

**VAT 관련 일상 상식**

| 상황 | 세금 여부 |
|------|-----------|
| 마트·편의점·식당 구매 | 포함 (현금영수증 요청 시 소득공제 가능) |
| 대형 슈퍼마켓 농축산물 | **면세** (가공 전 식품) |
| 의료·교육 서비스 | **면세** |
| 해외직구 (150달러 초과) | 관세 + VAT 별도 부과 |
| 영수증 없이 현금 결제 | 소득공제 불가 → 현금영수증 반드시 챙길 것 |

### ④ 자동차 관련 세금

차를 사고 보유하고 운전할 때마다 세금이 발생합니다.

| 단계 | 세금 | 세율·금액 |
|------|------|-----------|
| **구입 시** | 취득세 | 차값의 7% (비영업용 승용차 기준) |
| **구입 시** | 개별소비세 | 출고가의 5% (2,000cc 초과) |
| **구입 시** | 교육세·부가세 | 개별소비세의 30%, 최종 합산가에 VAT 10% |
| **보유 시** | 자동차세 | 배기량 기준 (cc당 80~200원 × cc, 연 2회 납부) |
| **주유 시** | 교통·에너지·환경세 | 휘발유 1ℓ당 약 529원 고정세 포함 |

> 💡 **자동차세 연납 할인**: 1월에 1년치 선납하면 최대 5% 할인 혜택이 있습니다.

```
[예시] 2,000cc 국산 승용차 2,500만원 구입 시
  취득세: 2,500만원 × 7% = 175만원
  자동차세: 2,000cc × 200원 = 40만원/년
```

### ⑤ 부동산 관련 세금 — 사고·보유·팔 때

| 단계 | 세금 | 세율 |
|------|------|------|
| **살 때 (취득)** | 취득세 | 1주택 1~3%, 2주택 8%, 3주택 이상 12% |
| **보유 중** | 재산세 | 공시가격의 0.1~0.4% (매년 6·9월) |
| **보유 중** | 종합부동산세 (종부세) | 공시가 합산 9억원(1주택 12억) 초과 시 추가 부과 |
| **팔 때 (양도)** | 양도소득세 | 보유기간·주택수에 따라 6~82% |

**1세대 1주택 비과세 요건**

```
① 양도가액 12억원 이하
② 보유기간 2년 이상 (조정대상지역은 거주 2년 이상)
→ 위 조건 충족 시 양도소득세 전액 비과세
```

> 💡 **장기보유특별공제**: 3년 이상 보유 시 양도차익의 최대 80%까지 공제 (1주택 10년 이상 보유·거주 시).

### ⑥ 가족 간 돈 이체 — 증여세 주의

가족이라도 **일정 금액 이상 무상으로 돈·재산을 주면 증여세**가 발생합니다.

| 증여자 | 10년 합산 공제 한도 |
|--------|---------------------|
| 배우자 | **6억원** |
| 직계존속 (부모·조부모) | **5천만원** (미성년자 2천만원) |
| 직계비속 (자녀·손자녀) | **5천만원** |
| 기타 친족 (형제·자매 등) | **1천만원** |

```
[예시] 부모가 성인 자녀에게 8천만원 이체
  → 공제 5천만원 초과분 3천만원에 증여세 10% = 300만원 납부
  → 신고 기한: 증여일이 속하는 달의 말일부터 3개월 이내
```

> ⚠️ **주의**: 증여 사실을 숨기다 적발되면 가산세(무신고 20%, 납부지연 1일 0.022%)가 붙습니다. 금액이 크다면 반드시 신고하세요.

### ⑦ 금융 소득에 붙는 세금 — 이자·배당

| 소득 유형 | 세율 | 원천징수 여부 |
|-----------|------|---------------|

## F-A. 자산의 기본 개념

> 📺 [🎬 자산의 기본 개념](https://www.youtube.com/results?search_query=자산의+기본+개념+한국어)

> 📖 **Wikipedia**: [자산](https://ko.wikipedia.org/wiki/자산) · [재무상태표](https://ko.wikipedia.org/wiki/재무상태표) · [감가상각](https://ko.wikipedia.org/wiki/감가상각)

### ① 자산(資産)이란?

**자산(Asset)**은 과거 사건의 결과로 기업(또는 개인)이 **통제하고 있고, 미래에 경제적 이익이 유입될 것으로 기대되는 자원**입니다.

```
핵심 조건 (모두 충족해야 자산으로 인식):
  ① 통제(Control)    — 내가 사용하거나 처분할 수 있어야 함
  ② 경제적 효익      — 미래에 현금흐름이나 가치를 가져다줘야 함
  ③ 신뢰성 있는 측정 — 금액을 합리적으로 추정할 수 있어야 함

예시:
  ✅ 자산: 공장 건물, 현금, 매출채권, 특허권, 주식 보유
  ❌ 자산 아님: 계약 체결 전 예정 수주, 내부 개발 중인 브랜드 가치
```

### ② 자산의 분류 체계

**[분류 1] 유동성(환금성)에 따른 분류 — 재무상태표 기본 구조**

| 구분 | 기준 | 내용 | 예시 |
|------|------|------|------|
| **유동자산** (Current Assets) | **1년 이내** 현금화 가능 | 빠르게 쓰거나 팔 수 있는 자산 | 현금·예금, 매출채권, 재고자산, 단기금융상품 |
| **비유동자산** (Non-current Assets) | **1년 초과** 보유 | 장기 보유·운용 목적 자산 | 건물·기계, 토지, 투자주식, 특허권, 이연법인세 |

```
재무상태표(Balance Sheet) 왼쪽

        비유동자산 합계     1,100억

자산 총계                   1,600억
```

**[분류 2] 형태에 따른 분류**

| 구분 | 영문 | 특성 | 예시 |
|------|------|------|------|
| **유형자산** | Tangible Assets | 눈에 보이고 만질 수 있는 자산 | 토지, 건물, 기계, 차량, 설비 |

| 인플레이션 | 가격 상승 시 함께 오름 | 명목가치 고정 시 실질가치 하락 |
| 퀀트 활용 | DCF 밸류에이션 기반 | 가격·수익률 모델링 기반 |

### ③ 자산 = 부채 + 자본 (회계 등식)

회계의 가장 기본 원칙입니다. **자산은 어딘가에서 조달한 것**이며, 조달 원천이 부채(빌린 돈)와 자본(내 돈)입니다.

```
자산 (Assets) = 부채 (Liabilities) + 자본 (Equity)
─────────────────────────────────────────────────────
내가 가진 것 = 남한테 빌린 것 + 내 것

예시:
  집을 5억에 샀을 때:
    자산 5억 = 대출(부채) 3억 + 내 돈(자본) 2억

  회사가 자산 1,600억을 보유할 때:
    자산 1,600억 = 부채(차입금·채권 등) 900억 + 자본(주주 출자·유보이익) 700억
```

| 항목 | 정의 | 투자자에게 의미 |
|------|------|---------------|
| **자산(A)** | 기업이 보유한 모든 경제적 자원 | 기업이 뭘 갖고 있는가 |
| **부채(L)** | 타인에게 갚아야 할 의무 | 얼마를 빌렸는가 |
| **자본(E = A - L)** | 주주의 몫 (순자산) | 실질적인 주주 가치 |

> 💡 **순자산(Net Asset) = 자본 = 자산 - 부채**. 주가순자산비율(PBR)의 기준이 되는 값입니다. `PBR = 시가총액 ÷ 순자산`

### ④ 자산의 가치 측정 방법

같은 자산이라도 **어떤 기준으로 측정하느냐**에 따라 금액이 달라집니다.

| 측정 기준 | 영문 | 정의 | 특징 |
|---------|------|------|------|
| **취득원가** | Historical Cost | 처음 살 때 지불한 금액 | 안정적이나 현재 가치 반영 못함 |
| **공정가치** | Fair Value | 현재 시장에서 거래될 합리적 가격 | 현실적이나 변동성 있음 |
| **장부가치** | Book Value | 취득원가 - 누적 감가상각 - 손상차손 | 회계장부에 기록된 금액 |
| **시장가치** | Market Value | 실제 시장에서 거래되는 가격 | 주가·부동산 시세 등 |
| **청산가치** | Liquidation Value | 강제 매각 시 회수 가능 금액 | 장부가치보다 낮을 때가 많음 |

```python
def asset_values(acquisition_cost: float, useful_life: int, years_used: int,
                 market_price: float) -> dict:
    """유형자산 가치 비교 (정액법 감가상각 기준)"""
    annual_depreciation = acquisition_cost / useful_life
    accumulated_depreciation = annual_depreciation * years_used
    book_value = acquisition_cost - accumulated_depreciation
    unrealized_gain_loss = market_price - book_value  # 시장가치 - 장부가치

    return {
        "취득원가":          f"{acquisition_cost:,.0f}원",
        "연간 감가상각":      f"{annual_depreciation:,.0f}원",
        "누적 감가상각":      f"{accumulated_depreciation:,.0f}원",
        "장부가치 (Book)":    f"{book_value:,.0f}원",
        "시장가치 (Market)":  f"{market_price:,.0f}원",
        "미실현 평가손익":    f"{unrealized_gain_loss:+,.0f}원",
    }

# 예시: 10억짜리 건물, 내용연수 40년, 10년 사용, 현재 시장가 13억
result = asset_values(1_000_000_000, 40, 10, 1_300_000_000)
for k, v in result.items():
    print(f"  {k}: {v}")
```

```
  취득원가:          1,000,000,000원
  연간 감가상각:       25,000,000원
  누적 감가상각:      250,000,000원
  장부가치 (Book):    750,000,000원
  시장가치 (Market): 1,300,000,000원
  미실현 평가손익:   +550,000,000원  ← 장부가치보다 5.5억 높음
```

### ⑤ 감가상각 (Depreciation) — 자산 가치가 줄어드는 것을 인식하는 방법

**유형자산**은 사용할수록 노후화되므로, 그 가치 감소를 비용으로 인식합니다.

```
건물 10억, 내용연수 40년 (정액법)

  구입 시:   장부가치 10억
  10년 후:   장부가치 7.5억 (2.5억 감가)
  20년 후:   장부가치 5억   (5억 감가)
  40년 후:   장부가치 0원   (완전 상각)

매년 2,500만원씩 손익계산서에 '감가상각비' 비용 계상
→ 당기순이익 감소 BUT 실제 현금 유출은 없음 (비현금 비용)
```

| 감가상각 방법 | 원리 | 특성 |
|------------|------|------|
| **정액법** (Straight-line) | 매년 동일한 금액 상각 | 예측 쉬움, 국내 가장 일반적 |
| **정률법** (Declining Balance) | 장부가치의 일정 % 상각 → 초기에 더 많이 비용 인식 | 초기 이익 작고, 후기 이익 큼 |
| **생산량비례법** | 실제 생산량에 비례해 상각 | 기계 등 사용량 측정 가능한 자산에 적합 |

> 💡 **EBITDA**: 세전이익(E) + 이자(I) + 세금(T) + **감가상각(DA)**. 감가상각이 클수록 EBITDA와 순이익의 차이가 커집니다. 장치 산업(반도체·통신·항공)은 EBITDA로 수익성을 비교하는 이유가 여기에 있습니다.

### ⑥ 개인의 자산 vs 법인의 자산

| 구분 | 개인 자산 | 법인 자산 |
|------|---------|---------|
| 소유 주체 | 자연인 본인 | 법인 (주주와 분리) |
| 부채 책임 | 개인 전 재산 (무한책임, 일반적) | 법인 자산 한도 (유한책임) |
| 세금 | 보유: 재산세·종합부동산세 / 처분: 양도소득세 | 보유: 법인세 / 처분: 법인세에 포함 |
| 상속·증여 | 상속세·증여세 직접 부과 | 법인 주식 이전으로 간접 이전 가능 |
| 자산 감가상각 | 사업소득 있는 경우 일부 인정 | 세법상 감가상각 전액 손금 처리 가능 |

```python
# 자산 분류 체계 정리
asset_classification = {
    "유동자산": {
```

## I. ISA — 개인종합자산관리계좌 (Individual Savings Account)

> 📺 [🎬 ISA 개인종합자산관리계좌](https://www.youtube.com/results?search_query=ISA+개인종합자산관리계좌+한국어)

> 📖 **Wikipedia**: [개인종합자산관리계좌](https://ko.wikipedia.org/wiki/개인종합자산관리계좌)

### ① ISA란?

**ISA(Individual Savings Account)**는 하나의 계좌 안에 예금·펀드·ETF·ELS·리츠 등 **다양한 금융상품을 담아 운용**하면서 **세금 혜택**을 받는 절세 투자 계좌입니다.

```
일반 계좌:
  펀드 A 이익 +100만원 → 세금 15.4만원
  펀드 B 손실 -50만원  → 손실 공제 없음
  실수령: 100 − 15.4 − 50 = 34.6만원

ISA 계좌:
  펀드 A 이익 +100만원
  펀드 B 손실 -50만원
  손익 통산 순이익 = 50만원
  비과세 한도 200만원 → 전액 비과세 (세금 0원!)
  실수령: 50만원
```

### ② ISA 유형·가입 조건·혜택 비교

| 구분 | 일반형 | 서민형 | 농어민형 |
|------|--------|--------|---------|
| 가입 대상 | 19세 이상 거주자 (근로·사업소득자) | 총급여 5천만원 또는 종합소득 3,800만원 이하 | 농어민 (소득 무관) |
| 비과세 한도 | **200만원** | **400만원** | **400만원** |
| 초과분 과세 | 9.9% **분리과세** | 9.9% 분리과세 | 9.9% 분리과세 |
| 의무 가입 기간 | **3년** | **3년** | **3년** |
| 연간 납입 한도 | **2,000만원** | 2,000만원 | 2,000만원 |
| 총 납입 한도 | **1억원** | 1억원 | 1억원 |

### ③ ISA 투자 가능 상품

| 상품 | 비고 |
|------|------|
| **예금·적금** | 확정 이자 |
| **국내 상장 ETF** | 주식·채권·리츠·원자재 ETF |
| **공모펀드** | 국내·해외 주식형·채권형 |
| **ELS·DLS** | 파생결합증권 |
| **리츠(REITs)** | 부동산투자신탁 |

> ⚠️ 해외 주식 직접 투자는 ISA에서 불가. ETF로 우회 투자해야 합니다.

### ④ ISA 세금 혜택 계산

```python
def isa_tax_benefit(profit: int, loss: int, account_type: str = "일반형") -> dict:
    net_profit = profit - loss
    tax_free_limit = 4_000_000 if account_type != "일반형" else 2_000_000
    if net_profit <= 0:
        return {"순이익": f"{net_profit:,}원", "세금": "0원"}
    taxable = max(0, net_profit - tax_free_limit)
    isa_tax = taxable * 0.099
    normal_tax = net_profit * 0.154
    return {
        "손익 통산 순이익": f"{net_profit:,}원",
        "비과세 한도": f"{tax_free_limit:,}원",
        "ISA 세금 (9.9%)": f"{isa_tax:,.0f}원",
        "일반 계좌 세금 (15.4%)": f"{normal_tax:,.0f}원",
        "절세 효과": f"{normal_tax - isa_tax:,.0f}원 절약",
    }

print("=== 일반형 ISA (이익 500만, 손실 100만) ===")
for k, v in isa_tax_benefit(5_000_000, 1_000_000, "일반형").items():
    print(f"  {k}: {v}")
```

### ⑤ ISA와 포트폴리오 연관성

```
계좌 유형별 자산 배분 최적화

  ISA 계좌         → 채권 ETF, 배당 ETF, ELS, 예금
                      (이자·배당 세금 절약 극대화)

  연금저축·IRP    → 장기 성장 자산 (주식형 ETF)
                      (납입 시 세액공제 + 과세 이연)

  일반 계좌       → 성장주, 해외 주식 직접 투자
                      (국내 주식 매매차익 비과세)
```

| 계좌 | 세금 혜택 | 의무 기간 | 적합 자산 |
|------|---------|---------|---------|
| **ISA** | 손익 통산 + 비과세 200/400만원 + 9.9% 분리과세 | 3년 | 채권 ETF, 배당 ETF |
| **연금저축** | 납입 세액공제 16.5%/13.2% + 과세 이연 | 55세 | 장기 주식 ETF |

## J. 종합소득세 vs 금융소득종합과세 — 차이와 연계

> 📺 [🎬 종합소득세 vs 금융소득종합과세 차이와](https://www.youtube.com/results?search_query=종합소득세+vs+금융소득종합과세+차이와+연계+한국어)

### ① 종합소득세(종소세)란?

**종합소득세**는 개인이 1년 동안 **모든 소득을 합산**해서 납부하는 세금입니다.

합산되는 소득의 종류:

| 소득 유형 | 예시 |
|---------|------|
| **근로소득** | 월급, 보너스 |
| **사업소득** | 자영업·프리랜서 수입 |
| **이자소득** | 예금 이자, 채권 이자 |
| **배당소득** | 주식 배당금, 펀드 분배금 |
| **연금소득** | 국민연금, 사적 연금 |
| **기타소득** | 강의료, 원고료, 경품 등 |
| **임대소득** | 부동산 임대 수입 |

> 💡 **이자소득 + 배당소득 = 금융소득**. 이 금융소득이 연 **2,000만원을 초과**하면 종합과세 대상이 됩니다.

### ② 금융소득 분리과세 vs 종합과세

| 구분 | 분리과세 | 종합과세 |
|------|---------|---------|
| 대상 | 금융소득 연 2,000만원 **이하** | 금융소득 연 2,000만원 **초과분** |
| 세율 | **15.4%** 원천징수로 종결 | 다른 소득과 합산 → 최고 49.5% |
| 신고 | 원천징수로 끝 (별도 신고 불필요) | **5월 종합소득세 신고 필수** |
| 유리한 경우 | 금융소득 적을 때, 다른 소득 높을 때 | 다른 소득이 없거나 낮을 때 |

```
[금융소득 종합과세 예시]

연봉 8,000만원 직장인 (종합세율 35%)
이자+배당 3,000만원 수령 시:

  원천징수 이미 뗀 금액:
    3,000만원 × 15.4% = 462만원

  2,000만원 초과분 = 1,000만원 → 종합과세
  기존 소득 8,000만원 + 초과 금융소득 1,000만원
  = 9,000만원 → 세율 35% 적용

  추가 납부세액 = 1,000만원 × (35% - 15.4%) = 196만원 추가 납부

  → 고소득자일수록 금융소득이 많으면 세 부담 급증!
```

### ③ 금융소득 종합과세 회피 절세 전략

| 전략 | 방법 |
|------|------|
| **ISA 활용** | 이자·배당을 ISA 계좌 안에 담아 운용 → 금융소득 합산 제외 |
| **연금 전환** | 이자·배당 대신 연금 형태로 수령 → 연금소득 분리 과세 |
| **배우자 분산** | 금융자산을 배우자에게 증여해 각자 2,000만원 이하 유지 |
| **비과세 상품** | 장기저축성보험, 비과세 펀드, 농어촌특별세 면제 상품 |
| **해외 계좌 이용** | 해외 주식 매매차익은 양도소득(분류과세) → 금융소득 미포함 |

### ④ 과세·비과세·분리과세 개념 정리

| 구분 | 의미 | 예시 |
|------|------|------|
| **과세** | 세금을 내야 하는 소득 | 대부분의 이자·배당·근로소득 |
| **비과세** | 법에서 세금 면제 | ISA 비과세 한도 내, 장기저축성보험 만기 수령 |
| **분리과세** | 다른 소득과 합산하지 않고 낮은 세율로 끝냄 | ISA 초과분 9.9%, 금융소득 2천만원 이하 15.4% |
| **종합과세** | 다른 소득과 합산해 누진세율 적용 | 금융소득 2천만원 초과분 |
| **원천징수** | 지급자가 미리 세금을 떼고 지급 | 이자·배당 지급 시 증권사·은행이 15.4% 공제 |

```python
def financial_income_tax(
    financial_income: int,
    other_income: int,
    isa_income: int = 0
) -> dict:
    """
    financial_income: 이자+배당 합계 (원)
    other_income    : 근로·사업 등 기타 소득 합계
    isa_income      : ISA 계좌 내 이익 (금융소득 합산 제외)
    """
    # 구간별 소득세율 (지방소득세 포함)
    brackets = [
        (14_000_000,   0.066),
        (50_000_000,   0.165),
        (88_000_000,   0.264),
        (150_000_000,  0.385),
        (300_000_000,  0.418),
        (500_000_000,  0.44),
        (1_000_000_000, 0.462),
        (float('inf'), 0.495),
    ]
    def calc_tax(income):
        tax, prev = 0, 0
        for limit, rate in brackets:
            if income <= prev:
                break
            taxable = min(income, limit) - prev
            tax += taxable * rate
            prev = limit
        return tax

    # 금융소득 2,000만원 이하: 분리과세 15.4%로 종결
    threshold = 20_000_000
    withholding = min(financial_income, threshold) * 0.154
    excess = max(0, financial_income - threshold)

    if excess == 0:
        return {"분리과세 세금": f"{withholding:,.0f}원", "종합과세 추가": "없음"}

    # 초과분 종합과세
    total_income = other_income + excess
    gross_tax = calc_tax(total_income)
    base_tax = calc_tax(other_income)
    additional_tax = gross_tax - base_tax - withholding * (excess / financial_income)

    return {
        "금융소득": f"{financial_income:,}원",
        "ISA 제외 금액": f"{isa_income:,}원",
        "원천징수 (15.4%)": f"{withholding:,.0f}원",
        "종합과세 추가 납부": f"{max(0, additional_tax):,.0f}원",
    }

# 연봉 8천만원, 이자+배당 3천만원
result = financial_income_tax(30_000_000, 80_000_000)
for k, v in result.items():
    print(f"  {k}: {v}")
```

## K. 과세·비과세·서민·일반인 관련 세금 정책 정리

### ① 소득 수준별 세금 차등 정책의 원리

한국 세금 제도는 **응능 부담 원칙(Ability-to-Pay)**을 기반으로 합니다.

> 소득이 많을수록 더 많이 내야 공평하다는 원칙.
> → 소득세 누진세율(6%~45%), 법인세 누진세율(9%~24%)

### ② 서민·저소득자 세금 혜택

| 혜택 | 대상 | 내용 |
|------|------|------|
| **근로장려금(EITC)** | 총소득 3,800만원 이하 근로자·자영업자 | 근로 의욕 고취 목적 현금 지원 |
| **서민형 ISA** | 총급여 5천만원 또는 종합소득 3,800만원 이하 | 비과세 한도 400만원 (일반형의 2배) |
| **소득공제 장기펀드** | 총급여 5천만원 이하 | 납입액 40% 소득공제 |
| **농어촌특별세 면제** | 농어민 | 금융투자 일부 비과세 |
| **기초공제·인적공제** | 전체 납세자 | 본인 150만원 + 부양가족 1인당 150만원 공제 |

### ③ 비과세 금융상품 종류

| 상품 | 비과세 조건 |
|------|-----------|
| **장기저축성보험** | 10년 이상 유지, 매월 150만원 이하 납입 |
| **비과세 종합저축** | 65세 이상·장애인 등, 원금 5천만원까지 이자 비과세 |
| **ISA 비과세 한도** | 만기 해지 시 200만원(일반형)/400만원(서민형) |
| **농어민 비과세 예탁금** | 농협·수협 조합원, 3천만원까지 이자 비과세 |
| **해외 주식 매매차익** | 연 250만원 이하 → 비과세 (초과 시 22% 양도소득세) |

#### 🔑 자본시장과 금융투자업에 관한 법률 (자본시장법)

- **언제 만들어졌나요?** 2009년 시행. 주식·채권·펀드·파생상품 등 거의 모든 금융투자 상품과 회사를 하나의 법으로 묶은 큰 법이에요.
- **쉽게 말하면?** 투자 시장의 '게임 규칙책'이에요.

| 자본시장법이 정하는 것 | 쉬운 설명 |
|----------------------|---------|
| **투자 위험 고지 의무** | 금융사가 상품을 팔 때 "이 상품은 손해가 날 수 있어요"라고 반드시 알려야 함 |
| **적합성 원칙** | 노인·초보에게 위험한 상품 함부로 팔면 안 됨 — 고객에게 맞는 상품만 권해야 함 |
| **불공정 거래 금지** | 내부자 거래, 시세조종, 거짓 정보 유포 금지 |
| **공시 의무** | 상장회사는 중요한 정보를 DART에 공개해야 함 |
| **인가·등록제** | 증권사, 자산운용사 등은 금융위원회 허가 없이 영업 불가 |

```
🎮 게임 비유:
   "스타트 버튼을 누르기 전에 게임 설명서를 줘야 한다"
   → 금융상품도 가입 전 필수 설명 의무!

   "치트키(불법 프로그램) 사용 금지"
   → 내부 정보 이용한 매매도 불법!
```

> 💡 **내부자 거래(Insider Trading)란?** 예를 들어 삼성전자 직원이 "다음 달 깜짝 실적이 나온다"는 비밀 정보를 미리 알고 주식을 사는 것. 일반 투자자보다 불공평한 이득이라서 **5년 이하 징역 또는 이득액의 3~5배 벌금**형을 받아요.

#### 🔑 금융소비자 보호에 관한 법률 (금융소비자보호법, 2021년 시행)

##### ① 제정 배경

- **대규모 불완전판매 사건**: DLS·라임·옵티머스 등 고난도 상품 판매 과정에서 설명 부족, 적합성 위반 문제가 반복됐습니다.
- **업권별 규제 파편화**: 은행·증권·보험·여전업 규제가 분절돼 동일한 판매행위라도 보호 수준이 달랐습니다.
- **소비자 중심 전환 필요**: "팔고 끝" 관행에서 "판매 전·중·후 책임" 체계로 전환하기 위해 단일법 제정이 추진됐습니다.

##### ② 금융소비자보호법 개관

| 항목 | 내용 |
|------|------|
| **법 목적** | 금융소비자 권익 증진, 금융상품 판매질서 확립, 신뢰 회복 |
| **적용 대상** | 예금성·대출성·보장성·투자성 금융상품 전반 |
| **핵심 체계** | 6대 판매원칙 + 소비자 권리 + 위반 시 강력 제재 |
| **감독 구조** | 금융위 정책·제도, 금감원 검사·분쟁조정 연계 |

##### ③ 금융소비자보호법 주요내용

| 핵심 규정 | 내용 | 쉬운 설명 |
|---------|------|---------|
| **6대 판매 원칙** | 적합성·적정성·설명의무·불공정행위금지·부당권유금지·광고규제 | 상품 팔 때 지켜야 할 6가지 약속 |

| **위법계약 해지권** | 판매 규정 위반 시 5년 이내 계약 해지 요청 가능 | 잘못 팔면 소비자가 해지 가능 |
| **손해배상 입증책임 전환** | 금융사가 "우리 잘못 아니다"를 입증해야 함 | 소비자 보호 강화 |

| **징벌적 과징금·제재 강화** | 위반행위에 과징금·과태료·임직원 제재 부과 가능 | 규정 위반 비용을 크게 높여 억제 |


#### 🔑 예금자 보호법

- **쉽게 말하면?** 은행이 망해도 내 돈 일부를 국가(예금보험공사)가 돌려주는 제도예요.

```
🏦 예시:
   A은행에 3,000만원 예금
   → A은행 갑자기 파산!
   → 예금보험공사가 최대 5,000만원까지 대신 지급 ✅

   5,000만원 초과분은 돌려받지 못할 수 있어요 ⚠️
```

| 보호 되는 상품 | 보호 안 되는 상품 |
|-------------|---------------|
| 예금·적금·부금·예탁금 | 주식·채권·펀드·ELS |

> 💡 **1인당 5,000만원 한도**는 같은 금융기관 내 모든 계좌 합산이에요. 1억원이 있다면 두 은행에 5,000만원씩 나눠 예치하면 전액 보호!

#### 🔑 상법 (商法) — 회사의 운영 규칙

- **쉽게 말하면?** 회사를 만들고, 운영하고, 없애는 모든 규칙을 담은 법이에요.

```
🏗️ 집 짓기 비유:
   건축법 없이 집을 지으면? → 마음대로 짓다가 무너짐
   상법 없이 회사 운영하면? → 사장 마음대로 해서 주주·직원 피해 발생
```

| 상법 주요 규정 | 내용 |
|-------------|------|
| **이사의 충실의무** | 이사는 회사와 주주를 위해 일해야 함 (개인 이익 먼저 X) |
| **이사의 선관주의의무** | 이사는 '평균적으로 주의 깊은 전문가'처럼 행동해야 함 |
| **주주총회 결의 사항** | 합병·분할·정관 변경·이사 선임은 주주 동의 필수 |
| **소수주주권** | 주식 1% 이상 보유 시 이사 해임 청구, 회계장부 열람 가능 |
| **배당 규제** | 이익 없으면 배당 불가, 채권자 보호 최우선 |

> 💡 **이사의 의무를 어기면?** 주주가 이사를 상대로 **주주대표소송**을 제기해 회사 손해를 배상받을 수 있어요.

### ③ 투자 관련 세금 법률 🧾

#### 🔑 국세기본법 — 세금 절차의 기본 규칙

```
국세기본법은 "세금을 어떤 절차로 부과·징수·환급할지"를 정한 공통 규칙입니다.

쉬운 말로:
  소득세법·법인세법이 "무슨 세금을 내는지"를 정한다면,
  국세기본법은 "언제, 어떻게 내고 다투는지"를 정하는 절차법이에요.
```

| 핵심 항목 | 내용 | 투자자 관점 |
|---------|------|-----------|
| **신고·납부 기한** | 법정 신고기한 내 신고·납부 의무 | 해외주식 양도차익은 보통 5월 확정신고 |
| **가산세** | 무신고·과소신고·납부지연 시 추가 부담 | "늦게 내면 원금+추가비용" 구조 |
| **국세부과 제척기간** | 일반적으로 5년(사기·부정은 10년 등) | 과거 거래도 일정 기간 내 추징 가능 |
| **경정청구·불복** | 과다납부 시 환급 청구, 과세 불복 절차 보장 | 투자자 권리구제 통로 |

> 💡 핵심: "세금 종류"만 아는 것보다 **기한·증빙·신고 절차**를 지키는 것이 실제 절세와 리스크 관리에 더 중요합니다.

#### 🔑 소득세법 — 이자·배당·양도소득 과세의 본체

| 구분 | 과세 방식 | 기본 세율/구조 | 신고 포인트 |
|------|---------|--------------|-----------|
| **이자소득** | 원천징수 후 분리과세(원칙) | 14% + 지방소득세 1.4% = **15.4%** | 금융소득 합계 2,000만원 초과 시 종합과세 |
| **배당소득** | 원천징수 후 분리과세(원칙) | **15.4%** | 고배당·고금리 자산 비중 높을수록 종합과세 주의 |
| **국내 상장주식 양도소득** | 소액주주 원칙적 비과세 | 대주주·특정 비상장주식은 과세 | 대주주 기준·판정시점은 매년 최신 규정 확인 |
| **해외주식 양도소득** | 분류과세(별도 신고) | 연간 손익통산 후 기본공제 250만원, 초과분 **22%(지방세 포함)** | 다음 해 5월 확정신고·납부 |

```
투자자가 헷갈리는 핵심
  이자·배당 = "받을 때 원천징수"
  양도차익 = "연말 손익 계산 후 신고"

  즉, 돈 들어오는 시점의 과세 구조가 서로 다릅니다.
```

##### 납세의무의 성립(成立) 관점에서 보면

- **성립 시점(언제 세금 의무가 생기나?)**: 주식을 **매도 체결한 때(거래가 성사된 시점)** 납세의무가 성립합니다.
- **납부 방식(누가 실제로 내나?)**: 투자자가 따로 납부서를 작성하기보다, 보통 **증권사가 매도대금에서 자동 공제**해 원천적으로 처리합니다.

```
초등학생 비유
  중고 장난감을 친구에게 "팔기로 약속이 딱 된 순간"
  → "수수료를 내야 하는 약속"이 바로 생김(= 납세의무 성립)
  실제 돈 전달할 때는
  → 엄마가 수수료를 먼저 떼고 남은 돈을 주는 것과 비슷함(= 원천 공제)
```

#### 🔑 양도소득세 (Capital Gains Tax)

```
🏠 부동산을 팔 때는 보통 양도소득세가 과세되고,
📈 금융투자에서는 자산 종류·거주자 여부·시장(국내/해외)에 따라 과세 방식이 달라집니다.

🌍 해외 주식 양도소득세(대표 구조):
   1년 손익통산 후 순이익 - 250만원 기본공제
   × 22% (지방소득세 포함) = 납부세액

   예시: 해외 주식 이익 1,000만원
   → (1,000만 - 250만) × 22% = 165만원 세금
   → 다음 해 5월에 직접 신고·납부
```

#### 법인세법 — 회사가 내는 세금의 규칙

```
법인세법 = 회사(법인)가 언제, 얼마나 세금을 내야 하는지 정한 법

핵심 개념:
   익금(益金) = 세법상 수익 (회계상 수익 + 추가 과세항목)
   손금(損金) = 세법상 비용 (회계상 비용 중 세법이 인정하는 것)
   과세표준 = 익금 - 손금
   납부세액 = 과세표준 × 세율 - 세액공제
```

| 법인세법 중요 규정 | 쉬운 설명 |
|----------------|---------|
| **접대비 한도** | 밥값·선물비용은 일정 한도 내에서만 비용 인정 |
| **감가상각 규정** | 건물·기계는 정해진 방식대로만 비용 처리 가능 |
| **이월결손금** | 올해 손실을 최대 15년간 다음 이익에서 차감 가능 |
| **세액공제·감면** | R&D 투자, 설비 투자, 고용 창출 시 세금 감면 혜택 |

```python
# 법인세 세액공제 간단 예시
# 연구개발(R&D) 비용 세액공제 (중소기업 최대 25%)
rd_expense = 100_000_000  # 1억원 R&D 투자
tax_credit_rate = 0.25    # 중소기업 25%
tax_credit = rd_expense * tax_credit_rate

corporate_tax_before = 50_000_000  # 산출세액 5천만원
corporate_tax_after  = corporate_tax_before - tax_credit

print(f"R&D 투자액:    {rd_expense:>12,}원")
print(f"세액공제:      {tax_credit:>12,}원")
print(f"납부 법인세:   {max(0, corporate_tax_after):>12,}원")
# → 납부 법인세: 25,000,000원 (세금 절반으로 줄어듦!)
```

### ⑦ 법인 설립·운영 단계별 법적 절차 📋

```
[법인 설립 ~ 운영 흐름도]

① 정관 작성
   └── 회사 이름, 목적, 자본금, 주식 수 등 기재

② 자본금 납입
   └── 주주들이 주식 대금을 은행에 입금
   └── 최저 자본금 제한 없음 (단, 실무상 최소 100만원 이상)

③ 법인 등기 (법원 등기소)
   └── 법인 탄생! 이때부터 법적 인격 생김

④ 사업자 등록 (세무서)
   └── 세금 납부·사업 가능

⑤ 4대보험 취득 신고 (직원 고용 시)
   └── 국민연금·건강보험·고용보험·산재보험

⑥ 부가세·법인세 신고 (정기)
   └── 부가세: 연 2회 (1월, 7월)
   └── 법인세: 연 1회 (결산월 + 3개월 내)

⑦ 외부감사 (해당 시)
   └── 자산 120억 이상 등 일정 규모 이상이면 필수

⑧ 주주총회 (연 1회 이상)
   └── 재무제표 승인, 이사 선임, 배당 결정
```

```python
# 법인 설립 비용 간단 추정
def corporation_setup_cost(capital: int) -> dict:
    """
    주식회사 설립 시 초기 비용 추정 (2024년 기준)
    capital: 자본금 (원)
    """
    # 등록면허세: 자본금 × 0.4% (서울시 기준, 중과세율 3배 적용 시)
    registration_tax = capital * 0.004
    # 교육세: 등록면허세 × 20%
    education_tax = registration_tax * 0.2
    # 공증료 (공증인 방문 기준 대략)
    notarization_fee = 300_000
    # 법무사 수수료 (대략)
    agent_fee = 400_000

    total = registration_tax + education_tax + notarization_fee + agent_fee
    return {
        "자본금":           f"{capital:,}원",
        "등록면허세":       f"{registration_tax:,.0f}원",
        "교육세":           f"{education_tax:,.0f}원",
        "공증료(추정)":     f"{notarization_fee:,}원",
        "법무사 수수료(추정)": f"{agent_fee:,}원",
        "총 설립 비용(추정)":  f"{total:,.0f}원",
    }

for cap in [10_000_000, 50_000_000, 100_000_000]:
    result = corporation_setup_cost(cap)
    print(f"\n[자본금 {cap:,}원 설립 시]")
    for k, v in result.items():
        print(f"  {k}: {v}")
```

## H. 퀀트 분석에서 세금이 중요한 이유

> 📺 [🎬 퀀트 분석에서 세금이 중요한](https://www.youtube.com/results?search_query=퀀트+분석에서+세금이+중요한+이유+한국어)

| 분석 항목 | 세금 연관성 |
|-----------|-----------|
| EPS | 법인세 후 순이익 기준 → 실효세율 변화가 EPS 직접 영향 |
| FCF / DCF | 세후영업이익(NOPAT) 기반 → 법인세율 변동 시 밸류에이션 전체 변화 |
| 배당수익률 | 배당소득세 15.4% 차감 후 실질 수익 계산 필요 |
| PER 국가 간 비교 | 법인세율 차이 → 세후이익 달라짐 → 단순 비교 왜곡 |
| 이연법인세 | 갑자기 커지면 과세소득↔회계이익 간 큰 괴리 신호 |

```python
def effective_tax_rate(pretax: int, tax_expense: int) -> float:
    return tax_expense / pretax if pretax > 0 else float('nan')

# 예시
etr = effective_tax_rate(6_000_000_000_000, 1_200_000_000_000)
print(f"실효 법인세율: {etr*100:.1f}%")   # 20.0%
print(f"명목 법인세율: 24.0% → {(0.24-etr)*100:.1f}%p 절세 효과")
```
