**쉽게 이해하기**
- **차트 이미지 분류 응용** 개념은 `특징 만들기 → 규칙 학습 → 라벨 예측` 3단계로 보면 쉽습니다.
- 모델 성능은 정확도 하나만 보지 말고 오분류 사례를 같이 점검하세요.

**미니 예시**

## 오늘 내용 한눈에 정리

- **CNN**은 그림이나 차트에서 중요한 모양을 자동으로 찾는 딥러닝 모델입니다.
- **합성곱 층**은 작은 창으로 데이터를 훑어 특징(패턴)을 잡아냅니다.
- **풀링 층**은 중요한 정보만 남겨 계산을 가볍게 합니다.
- **1D CNN**은 시간 순서 데이터(주가 같은 시계열)에도 사용할 수 있습니다.
