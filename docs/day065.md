### 1. 클러스터별 종목 특성 분석

**쉽게 이해하기**
- **클러스터별 종목 특성 분석** 개념은 `특징 만들기 → 규칙 학습 → 라벨 예측` 3단계로 보면 쉽습니다.
- 모델 성능은 정확도 하나만 보지 말고 오분류 사례를 같이 점검하세요.

**미니 예시**
```python
# 클러스터별 종목 특성 분석 미니 예시
scores = [0.2, 0.7, 0.9, 0.4]
labels = ['매도' if s < 0.5 else '매수' for s in scores]
print('분류 결과:', labels)
```

- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
