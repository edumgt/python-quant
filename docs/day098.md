## 천천히 배우기 (초등학생도 이해할 수 있어요)

### 1. 클라우드(AWS/GCP) 또는 로컬 서버 배포

**쉽게 이해하기**
- **클라우드(AWS/GCP) 또는 로컬 서버 배포**는 `작업 실행 → 로그 기록 → 상태 알림` 루프를 만들면 운영이 쉬워집니다.
- 먼저 로컬에서 한 번 자동 실행해보고, 실패 로그가 남는지 확인한 뒤 배포하세요.

**미니 예시**
```python
# 클라우드(AWS/GCP) 또는 로컬 서버 배포 미니 예시
job_name = 'rebalance-daily'
status = 'success'
log = f'[{job_name}] 상태: {status}'
print(log)
if status != 'success':
    print('알림 전송: 운영자 확인 필요')
```
