### 5. 수학/통계 함수 (sum, mean, std, max)

**쉽게 이해하기**
- **수학/통계 함수 (sum, mean, std, max)** 개념은 작은 변수와 함수부터 직접 실행해보면 가장 빨리 익숙해집니다.
- 한 번에 길게 작성하기보다, 한 줄씩 출력 결과를 확인하면서 진행하세요.

**미니 예시**
```python
# 수학/통계 함수 (sum, mean, std, max) 미니 예시
name = 'quant'
scores = [80, 90, 100]
avg = sum(scores) / len(scores)
print(f'{name} 평균 점수: {avg:.1f}')
```

### 6. 실습: 주가 수익률 배열 연산

**쉽게 이해하기**
- **실습: 주가 수익률 배열 연산** 개념은 `일별 수익률 계산 → 누적 성과 확인 → 리스크 지표 비교`로 이해하면 쉽습니다.
- 먼저 작은 구간에서 계산이 맞는지 검증한 뒤, 전체 기간으로 확장하세요.

**미니 예시**
```python
# 실습: 주가 수익률 배열 연산 미니 예시
returns = [0.01, -0.02, 0.015, 0.005]
cum = 1.0
for r in returns:
    cum *= (1 + r)
print(f'누적수익률: {(cum - 1):.2%}')
```
