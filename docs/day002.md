### 1. 조건문: if, elif, else

**쉽게 이해하기**
- **조건문: if, elif, else** 개념은 작은 변수와 함수부터 직접 실행해보면 가장 빨리 익숙해집니다.
- 한 번에 길게 작성하기보다, 한 줄씩 출력 결과를 확인하면서 진행하세요.

**미니 예시**
```python
# 조건문: if, elif, else 미니 예시
name = 'quant'
scores = [80, 90, 100]
avg = sum(scores) / len(scores)
print(f'{name} 평균 점수: {avg:.1f}')
```

### 2. 비교 연산자 / 논리 연산자

**쉽게 이해하기**
- **비교 연산자 / 논리 연산자** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 비교 연산자 / 논리 연산자 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 3. 반복문: for 루프와 range()

**쉽게 이해하기**
- **반복문: for 루프와 range()** 개념은 작은 변수와 함수부터 직접 실행해보면 가장 빨리 익숙해집니다.
- 한 번에 길게 작성하기보다, 한 줄씩 출력 결과를 확인하면서 진행하세요.

**미니 예시**

- 한 번에 길게 작성하기보다, 한 줄씩 출력 결과를 확인하면서 진행하세요.

### 6. 리스트 컴프리헨션 (List Comprehension)

**쉽게 이해하기**

- 한 번에 길게 작성하기보다, 한 줄씩 출력 결과를 확인하면서 진행하세요.
