### 1. Risk-Parity 원리 (동등 리스크 기여)

**쉽게 이해하기**
- **Risk-Parity 원리 (동등 리스크 기여)** 개념은 `자산 비중 정하기 → 성과 계산 → 비중 조정` 흐름으로 보면 쉽습니다.
- 한 번에 복잡하게 최적화하지 말고 2개 자산 예제로 먼저 감을 잡으세요.

**미니 예시**
```python
# Risk-Parity 원리 (동등 리스크 기여) 미니 예시
weights = {'stock': 0.6, 'bond': 0.4}
asset_return = {'stock': 0.12, 'bond': 0.04}
portfolio_return = sum(weights[a] * asset_return[a] for a in weights)
print(f'기대수익률: {portfolio_return:.2%}')
```

### 4. 실습: Risk-Parity vs 평균분산 성과 비교

**쉽게 이해하기**
- **실습: Risk-Parity vs 평균분산 성과 비교** 개념은 `일별 수익률 계산 → 누적 성과 확인 → 리스크 지표 비교`로 이해하면 쉽습니다.
- 먼저 작은 구간에서 계산이 맞는지 검증한 뒤, 전체 기간으로 확장하세요.

**미니 예시**
```python
# 실습: Risk-Parity vs 평균분산 성과 비교 미니 예시
returns = [0.01, -0.02, 0.015, 0.005]
cum = 1.0
for r in returns:
    cum *= (1 + r)
print(f'누적수익률: {(cum - 1):.2%}')
```
