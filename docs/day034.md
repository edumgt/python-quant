### 1. K-Means 클러스터링 원리

**쉽게 이해하기**
- **K-Means 클러스터링 원리** 개념은 `특징 만들기 → 규칙 학습 → 라벨 예측` 3단계로 보면 쉽습니다.
- 모델 성능은 정확도 하나만 보지 말고 오분류 사례를 같이 점검하세요.

**미니 예시**
```python

labels = ['매도' if s < 0.5 else '매수' for s in scores]
print('분류 결과:', labels)
```

**쉽게 이해하기**
- **엘보우 방법(Elbow Method)으로 K 선택** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
