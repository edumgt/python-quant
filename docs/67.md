### 1. MACD(이동평균 수렴·발산) 계산 및 시그널

**쉽게 이해하기**
- **MACD(이동평균 수렴·발산) 계산 및 시그널** 개념은 `지표 계산 → 조건 비교 → 신호 해석` 흐름으로 접근하면 쉽습니다.
- 지표 하나만 보지 말고 거래량/추세와 함께 확인하는 습관이 중요합니다.

**미니 예시**
```python
# MACD(이동평균 수렴·발산) 계산 및 시그널 미니 예시
prices = [100, 102, 101, 105, 107]
ma3 = sum(prices[-3:]) / 3
signal = '매수 관찰' if prices[-1] > ma3 else '대기'
print('MA3:', round(ma3, 2), '| 신호:', signal)
```

### 2. 볼린저밴드 계산 및 스퀴즈 전략

**쉽게 이해하기**
- **볼린저밴드 계산 및 스퀴즈 전략** 개념은 `지표 계산 → 조건 비교 → 신호 해석` 흐름으로 접근하면 쉽습니다.
- 지표 하나만 보지 말고 거래량/추세와 함께 확인하는 습관이 중요합니다.

**미니 예시**
```python
# 볼린저밴드 계산 및 스퀴즈 전략 미니 예시
prices = [100, 102, 101, 105, 107]
ma3 = sum(prices[-3:]) / 3
signal = '매수 관찰' if prices[-1] > ma3 else '대기'
print('MA3:', round(ma3, 2), '| 신호:', signal)
```

### 3. 두 인디케이터 복합 시그널 생성

**쉽게 이해하기**
- **두 인디케이터 복합 시그널 생성** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 두 인디케이터 복합 시그널 생성 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 4. 실습: MACD + 볼린저밴드 전략 백테스트

**쉽게 이해하기**
- **실습: MACD + 볼린저밴드 전략 백테스트** 개념은 `일별 수익률 계산 → 누적 성과 확인 → 리스크 지표 비교`로 이해하면 쉽습니다.
- 먼저 작은 구간에서 계산이 맞는지 검증한 뒤, 전체 기간으로 확장하세요.

**미니 예시**
```python
# 실습: MACD + 볼린저밴드 전략 백테스트 미니 예시
returns = [0.01, -0.02, 0.015, 0.005]
cum = 1.0
for r in returns:
    cum *= (1 + r)
print(f'누적수익률: {(cum - 1):.2%}')
```
