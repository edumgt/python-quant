### 1. 연말 랠리(Santa Claus Rally) 분석

**쉽게 이해하기**
- **연말 랠리(Santa Claus Rally) 분석** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 연말 랠리(Santa Claus Rally) 분석 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

- 한 번에 길게 작성하기보다, 한 줄씩 출력 결과를 확인하면서 진행하세요.

**미니 예시**
```python
# 실습: 코스피/S&P500 계절성 패턴 Python 분석 미니 예시

scores = [80, 90, 100]
avg = sum(scores) / len(scores)
print(f'{name} 평균 점수: {avg:.1f}')
```
