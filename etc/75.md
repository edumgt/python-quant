### 1. Sharpe Ratio, MDD, Win Rate 종합 분석

**쉽게 이해하기**
- **Sharpe Ratio, MDD, Win Rate 종합 분석** 개념은 `일별 수익률 계산 → 누적 성과 확인 → 리스크 지표 비교`로 이해하면 쉽습니다.
- 먼저 작은 구간에서 계산이 맞는지 검증한 뒤, 전체 기간으로 확장하세요.

**미니 예시**
```python
# Sharpe Ratio, MDD, Win Rate 종합 분석 미니 예시
returns = [0.01, -0.02, 0.015, 0.005]
cum = 1.0
for r in returns:
    cum *= (1 + r)
print(f'누적수익률: {(cum - 1):.2%}')
```

### 2. 몬테카를로 시뮬레이션으로 전략 강건성 검증

**쉽게 이해하기**
- **몬테카를로 시뮬레이션으로 전략 강건성 검증** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 몬테카를로 시뮬레이션으로 전략 강건성 검증 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```
