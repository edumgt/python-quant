### 1. 신경망 구조 (입력층, 은닉층, 출력층)

**쉽게 이해하기**
- **신경망 구조 (입력층, 은닉층, 출력층)** 개념은 `특징 만들기 → 규칙 학습 → 라벨 예측` 3단계로 보면 쉽습니다.
- 모델 성능은 정확도 하나만 보지 말고 오분류 사례를 같이 점검하세요.

**미니 예시**
```python
# 신경망 구조 (입력층, 은닉층, 출력층) 미니 예시
scores = [0.2, 0.7, 0.9, 0.4]
labels = ['매도' if s < 0.5 else '매수' for s in scores]
print('분류 결과:', labels)
```

### 2. 활성화 함수 (ReLU, Sigmoid, Tanh, Softmax)

**쉽게 이해하기**
- **활성화 함수 (ReLU, Sigmoid, Tanh, Softmax)** 개념은 작은 변수와 함수부터 직접 실행해보면 가장 빨리 익숙해집니다.
- 한 번에 길게 작성하기보다, 한 줄씩 출력 결과를 확인하면서 진행하세요.

**미니 예시**
```python
# 활성화 함수 (ReLU, Sigmoid, Tanh, Softmax) 미니 예시
name = 'quant'
scores = [80, 90, 100]
avg = sum(scores) / len(scores)
print(f'{name} 평균 점수: {avg:.1f}')
```

### 3. 역전파(Backpropagation)와 가중치 업데이트

**쉽게 이해하기**
- **역전파(Backpropagation)와 가중치 업데이트** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 역전파(Backpropagation)와 가중치 업데이트 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 4. 드롭아웃(Dropout), 배치 정규화(Batch Normalization)

**쉽게 이해하기**
- **드롭아웃(Dropout), 배치 정규화(Batch Normalization)** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 드롭아웃(Dropout), 배치 정규화(Batch Normalization) 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 5. 실습: PyTorch/Keras MLP로 주가 방향 예측

**쉽게 이해하기**
- **실습: PyTorch/Keras MLP로 주가 방향 예측** 개념은 `입력(x)와 출력(y)의 관계 찾기`라고 생각하면 쉽습니다.
- 예측값만 보지 말고, 실제값과의 차이(오차)를 같이 확인하세요.

**미니 예시**
```python
# 실습: PyTorch/Keras MLP로 주가 방향 예측 미니 예시
x = [1, 2, 3, 4]
y = [2, 4, 6, 8]
slope = (y[-1] - y[0]) / (x[-1] - x[0])
next_y = y[-1] + slope
print('다음 값 예측:', next_y)
```

# Chapter 13. MLP Neural Network — 다층 퍼셉트론 신경망

> 💡 **쉽게 이해하기**: 사람의 뇌처럼 뉴런이 여러 층으로 연결된 신경망입니다. 입력이 들어오면 층을 거치며 변환되고, 틀린 예측만큼 오차를 역방향으로 전파(역전파)하여 가중치를 조금씩 수정하며 학습합니다.

---

### 핵심 개념

MLP(Multi-Layer Perceptron)는 **입력층 → 은닉층(들) → 출력층**으로 구성된 가장 기본적인 피드포워드 신경망입니다.

```
입력 특성 → [128] → [64] → [32] → 출력 클래스
               ReLU    ReLU   ReLU
```

### 역전파(Backpropagation)

- 예측 오차(손실)를 계산 → 그래디언트 역전파 → 가중치 업데이트
- `Adam` 옵티마이저: 학습률을 자동 조절하여 빠른 수렴

### 주요 파라미터

| 파라미터 | 설명 |
|--------|------|
| `hidden_layer_sizes` | 각 은닉층 뉴런 수 예: `(128, 64, 32)` |
| `activation` | 활성화 함수 (`relu`, `tanh`, `logistic`) |
| `solver` | 옵티마이저 (`adam`, `sgd`) |
| `max_iter` | 최대 에포크 수 |

### 손실 곡선 해석

- 손실이 **단조 감소** → 정상 학습
- **급격한 하락 후 정체** → 학습 완료 또는 수렴
- **진동** → 학습률이 너무 큼 (또는 소량 데이터)

### Scikit-learn MLP vs PyTorch/TensorFlow

| 항목 | sklearn MLP | PyTorch/TF |
|------|------------|------------|
| 난이도 | 쉬움 | 중간~어려움 |
| GPU 지원 | ❌ | ✅ |
| 유연성 | 제한적 | 매우 높음 |
| 교육 용도 | ✅ 개념 학습 | ✅ 실무 학습 |
