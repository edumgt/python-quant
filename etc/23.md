### 1. RNN(Recurrent Neural Network) 원리

**쉽게 이해하기**
- **RNN(Recurrent Neural Network) 원리** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# RNN(Recurrent Neural Network) 원리 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 2. 장기 의존성 문제와 기울기 소실

**쉽게 이해하기**
- **장기 의존성 문제와 기울기 소실** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# 장기 의존성 문제와 기울기 소실 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 3. LSTM(Long Short-Term Memory) 구조

**쉽게 이해하기**
- **LSTM(Long Short-Term Memory) 구조** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# LSTM(Long Short-Term Memory) 구조 미니 예시

```

### 4. GRU(Gated Recurrent Unit) 개요

**쉽게 이해하기**
- **GRU(Gated Recurrent Unit) 개요** 개념은 `입력값 확인 → 규칙 적용 → 결과 해석` 순서로 단계별로 보면 쉽습니다.
- 먼저 작은 데이터로 검증한 뒤 실제 데이터에 적용하세요.

**미니 예시**
```python
# GRU(Gated Recurrent Unit) 개요 미니 예시
values = [100, 103, 101, 106]
delta = values[-1] - values[0]
print('시작 대비 변화:', delta)
```

### 5. 실습: LSTM으로 주가 예측 모델 구현

**쉽게 이해하기**
- **실습: LSTM으로 주가 예측 모델 구현** 개념은 `입력(x)와 출력(y)의 관계 찾기`라고 생각하면 쉽습니다.
- 예측값만 보지 말고, 실제값과의 차이(오차)를 같이 확인하세요.

**미니 예시**
```python
# 실습: LSTM으로 주가 예측 모델 구현 미니 예시
x = [1, 2, 3, 4]
y = [2, 4, 6, 8]
slope = (y[-1] - y[0]) / (x[-1] - x[0])
next_y = y[-1] + slope
print('다음 값 예측:', next_y)
```

# Chapter 18. LSTM — 주가 수익률 시계열 예측

> 💡 **쉽게 이해하기**: 일반 신경망은 어제 배운 내용을 잊지만, LSTM은 "기억 창고(Cell State)"와 "게이트(Gate)"로 과거 정보를 선택적으로 기억합니다. 주가처럼 순서가 중요한 시계열에서 LSTM이 강력한 이유입니다. `LSTMPredictor.py`로 직접 실습하세요.

---

## 📌 이 챕터에서 배우는 것

| 개념 | 설명 |
|------|------|
| Forget Gate | 과거 정보 중 버릴 것 선택 (sigmoid → 0: 버림, 1: 유지) |
| Input Gate | 새 정보 중 기억할 것 선택 |
| Cell State | 장기 기억 저장소 |
| Output Gate | 현재 출력할 정보 선택 |
| 방향성 정확도 | 상승/하락 방향을 맞춘 비율 (퀀트 핵심 지표) |

> **Day037 연결** — `docs/23.md` 이론과 `src/LSTMPredictor.py` 실습을 함께 진행하세요.

## 🧱 모델 구조 (요약)

```
입력: (batch, 30, 1)        ← 30일 수익률 시퀀스 (피처 1개)
  │
  ▼ LSTM(hidden=64, layers=2) — 양방향 장기 의존성 학습
  ▼ 마지막 타임스텝 출력 (batch, 64)
  ▼ Linear(64→32) + ReLU
  ▼ Linear(32→1)
출력: (batch,)               ← 다음 날 수익률 예측값
```

---

## 📊 학습 결과 해석

| 지표 | 의미 |
|------|------|
| Val MSE Loss | 낮을수록 예측 오차 감소 |
| 방향성 정확도 > 0.52 | 무작위(0.50) 이상이면 통계적 유의미 |
| 과적합 | Train Loss << Val Loss → Dropout 증가, 데이터 확대 |

---

## ⚙️ 하이퍼파라미터 튜닝 팁

| 파라미터 | 권장 탐색 범위 | 영향 |
|---------|-------------|------|
| `seq_len` | 10 ~ 60 | 짧으면 최근 패턴, 길면 장기 추세 |
| `hidden_size` | 32 ~ 256 | 클수록 표현력↑, 과적합 위험↑ |
| `num_layers` | 1 ~ 3 | 깊을수록 복잡한 패턴 학습 |
| `dropout` | 0.1 ~ 0.5 | 과적합 조절 |

- **다변수 LSTM**: `input_size=N`으로 N개 팩터 동시 학습
