### 1. 차원 관점의 벡터·행렬 연산과 금융 데이터 해석

**쉽게 이해하기**
- 벡터는 `한 시점의 여러 자산 상태`, 행렬은 `여러 시점의 데이터 테이블`로 이해하면 쉽습니다.
- 예를 들어 `T x N` 수익률 행렬에서 `T`는 날짜 수, `N`은 자산 수입니다.

**미니 예시**
```python
# 5일 x 3자산 수익률 행렬 (T x N)
import numpy as np
R = np.array([
    [0.01, 0.00, -0.01],
    [0.02, -0.01, 0.00],
    [-0.01, 0.01, 0.01],
    [0.00, 0.02, -0.02],
    [0.01, -0.01, 0.00],
])
mu = R.mean(axis=0)  # 자산별 평균 수익률 벡터 (N,)
print("자산별 평균 수익률:", mu)
```

### 2. 선형시스템 및 해법 (np.linalg.solve, lstsq)

**쉽게 이해하기**
- 선형시스템 `Ax=b`는 `제약식 + 목표식`을 행렬로 표현한 형태입니다.
- 정사각행렬이면 `solve`, 과잉결정이면 `lstsq`(최소제곱)를 사용합니다.

**미니 예시**
```python
# 2자산 가중치 합=1, 목표수익률=0.01
import numpy as np
A = np.array([[1, 1], [0.02, 0.005]])
b = np.array([1.0, 0.01])
w = np.linalg.solve(A, b)
print("해:", w)
```

### 3. 포트폴리오 적용: 평균-분산 프레임

**쉽게 이해하기**
- 기대수익률은 `wᵀμ`, 분산은 `wᵀΣw`로 계산합니다.
- 행렬식으로 쓰면 자산 수가 늘어나도 계산 구조가 유지됩니다.

**미니 예시**
```python
# 분산-공분산 행렬 기반 포트폴리오 리스크
import numpy as np
w = np.array([0.5, 0.3, 0.2])
Sigma = np.array([
    [0.040, 0.010, 0.008],
    [0.010, 0.090, 0.015],
    [0.008, 0.015, 0.025],
])
risk = w @ Sigma @ w
print("포트폴리오 분산:", risk)
```

### 4. 랭크·공간 해석, 직교 투영, 베타 중립

**쉽게 이해하기**
- 랭크는 독립적인 정보량을 뜻하며, 낮으면 제약식이 중복될 수 있습니다.
- 직교 투영은 요인(시장 베타) 성분을 분리하는 핵심 도구입니다.
- 베타 중립은 `βᵀw = 0` 제약으로 시장 방향 노출을 줄입니다.

**미니 예시**
```python
# 베타 중립 제약 확인
import numpy as np
beta = np.array([1.2, 0.8, 0.5])
w = np.array([0.4, -0.3, -0.1])
print("포트폴리오 베타:", beta @ w)  # 0에 가까울수록 중립
```

### 5. 역행렬·결정식·조건수와 수치 안정성

**쉽게 이해하기**
- 역행렬은 해를 구하는 도구지만, 직접 역행렬 계산보다 `solve`가 안정적입니다.
- 결정식이 0에 가까우면 행렬이 특이(singular)할 수 있습니다.
- 조건수(condition number)가 크면 작은 데이터 오차가 큰 해 오차로 증폭됩니다.

**미니 예시**
```python
# 조건수 확인
import numpy as np
A = np.array([[1.0, 1.0], [1.0001, 1.0]])
print("det(A):", np.linalg.det(A))
print("cond(A):", np.linalg.cond(A))
```

### 6. 직교성과 QR 분해, 최소제곱

**쉽게 이해하기**
- QR 분해 `A=QR`는 선형회귀를 안정적으로 푸는 표준 방법입니다.
- Q의 열벡터는 서로 직교하여 수치적으로 더 안정적인 계산이 가능합니다.

**미니 예시**
```python
import numpy as np
X = np.array([[1, 1], [1, 2], [1, 3]], dtype=float)
y = np.array([1.0, 2.1, 2.9])
Q, R = np.linalg.qr(X)
beta = np.linalg.solve(R, Q.T @ y)
print("회귀계수:", beta)
```

### 7. 대칭 행렬과 고유값 분석 (리스크 분해)

**쉽게 이해하기**
- 공분산 행렬은 대칭 행렬이며, 고유값은 위험의 크기, 고유벡터는 위험 방향을 나타냅니다.
- 큰 고유값 몇 개가 전체 위험의 대부분을 설명하면 차원 축소(PCA)에 유리합니다.

**미니 예시**
```python
import numpy as np
Sigma = np.array([
    [0.040, 0.010, 0.008],
    [0.010, 0.090, 0.015],
    [0.008, 0.015, 0.025],
])
eigvals, eigvecs = np.linalg.eigh(Sigma)  # 대칭행렬 전용
print("고유값:", eigvals)
```

### 8. 실습: 최소분산 문제의 제약식(가능해) 이해 + 베타 중립 포트폴리오

**쉽게 이해하기**
- 최소분산 목적함수와 선형 제약(가중치 합=1, 베타중립=0)을 함께 풀어 실전형 문제로 연결합니다.
- 아래 미니 예시는 최적화 자체보다, 제약을 만족하는 가능한 가중치를 구성·검증하는 데 초점을 둡니다.
- 먼저 3자산 소규모 예제로 검증하고, 이후 자산 수를 늘리세요.

**미니 예시**
```python
import numpy as np
Sigma = np.array([
    [0.040, 0.010, 0.008],
    [0.010, 0.090, 0.015],
    [0.008, 0.015, 0.025],
])
beta = np.array([1.2, 0.8, 0.5])
mu = np.array([0.012, 0.010, 0.009])

# 단순 예시 해 (롱숏 허용, 가중치 합=1 + 베타중립을 만족하는 "가능해" 예시)
# 주의: 이 값은 최소분산 "최적해"가 아니라, 제약을 만족하는 후보를 손계산으로 만든 예시입니다.
# 유도 아이디어:
# (1) 계산 단순화를 위해 w3=1을 임의로 고정하면 합 제약 w1+w2+w3=1에서 w1+w2=0
# (2) 베타중립 1.2w1+0.8w2+0.5w3=0에 w3=1 대입하면 1.2w1+0.8w2=-0.5
# (3) 두 식을 풀면 w1=-1.25, w2=1.25
w = np.array([-1.25, 1.25, 1.00])
print("합 제약:", w.sum())
print("베타 제약:", beta @ w)
print("분산:", w @ Sigma @ w)
print("기대수익률:", w @ mu)
```

## 오늘 내용 한눈에 정리

- **벡터·행렬**은 금융 데이터를 구조적으로 표현하는 언어입니다.
- **선형시스템과 랭크**는 제약식의 해 존재성과 유일성을 판단하게 해줍니다.
- **직교 투영·QR 분해**는 회귀/요인모델 계산을 더 안정적으로 만듭니다.
- **조건수와 고유값 분석**은 수치불안정 구간과 핵심 리스크 요인을 찾게 해줍니다.
- **베타 중립 제약**은 시장 방향성 노출을 줄이는 실전 포트폴리오 기법입니다.

## 용어 사전 (초등학생도 이해하기 쉽게)

| 용어 | 한자 | 영어 | 쉬운 설명 |
|---|---|---|---|
| 벡터 | 向量 | Vector | 숫자를 한 줄로 모은 화살표 |
| 행렬 | 行列 | Matrix | 숫자를 표처럼 모은 것 |
| 랭크 | 階數 | Rank | 서로 다른 정보가 몇 개인지 |
| 직교 | 直交 | Orthogonal | 서로 딱 직각이라 겹치지 않음 |
| 투영 | 投影 | Projection | 그림자를 벽에 비추듯 성분을 분리 |
| 결정식 | 行列式 | Determinant | 행렬이 뒤집힐 수 있는지 알려주는 값 |
| 조건수 | 條件數 | Condition Number | 계산이 흔들리기 쉬운 정도 |
| 고유값 | 固有値 | Eigenvalue | 데이터가 가장 크게 흔들리는 크기 |
